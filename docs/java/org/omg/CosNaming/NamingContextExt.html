    public interface NamingContextExt
extends NamingContextExtOperations, NamingContext, IDLEntity
    NamingContextExt是NamingContext的NamingContext ，它包含一组名称绑定，其中每个名称是唯一的，并且是可互操作命名服务的一部分。 不同的名称可以在相同或不同的上下文中同时绑定到一个对象。 使用NamingContextExt ，您可以使用基于URL的名称进行绑定和解析。

    见CORBA COS Naming Specification.
