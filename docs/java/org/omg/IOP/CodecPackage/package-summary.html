Package org.omg.IOP.CodecPackage Description

该包是从IOP :: Codec IDL接口定义生成的。 它包含IOP :: Codec中指定的IDL异常的Java表示（作为便携式拦截器规范的一部分）。 有关详细信息，请参阅OMG规范。
