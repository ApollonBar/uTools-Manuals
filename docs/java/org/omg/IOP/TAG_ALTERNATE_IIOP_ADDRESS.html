      • value

        static final int value
        在同一个对象密钥用于多个互联网位置的情况下，下列标准IOR组件被定义为IIOP 1.2版中的支持。

        TAG_ALTERNATE_IIOP_ADDRESS组件具有相关联的值类型： 

           struct { string HostID, short Port };
        编码为CDR封装。

        TAG_ALTERNATE_IIOP_ADDRESS组件类型的零个或多个实例可能包含在版本1.2 TAG_INTERNET_IOP配置文件中。 除了在配置文件的主体中表示的主机和端口地址之外，这些替代地址中的每一个可以由客户端orb使用。 在其中一个或更多的情况下TAG_ALTERNATE_IIOP_ADDRESS组分存在于一个TAG_INTERNET_IOP资料，没有用的顺序是由IIOP的版本1.2规定的。
