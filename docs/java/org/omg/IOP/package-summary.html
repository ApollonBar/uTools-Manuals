Package org.omg.IOP Description

该包包含OMG文件“通用对象请求代理：体系结构和规范 ” http://cgi.omg.org/cgi-bin/doc?formal/99-10-07第13.6节中指定的IOP模块。 有关详细信息，请参阅OMG规范。

请注意，我们不提供规范中IOP模块的所有部分。 我们只提供在公共API中引用为返回值或参数类型的部分，最值得注意的是便携式拦截器。

包装规格

有关Java [tm] Platform，Standard Edition 6 ORB符合的官方规范支持部分的精确列表，请参阅Official Specifications for CORBA support in Java[tm] SE 6 。
