        static final int value
        在现实世界中，能够识别对象引用来自ORB的特定类型，解决该特定ORB的问题，或者利用共享的效率，在现实世界中通常是有用的。

        TAG_ORB_TYPE组件具有无符号长（Java long）类型的相关值，编码为CDR封装，指定由OMG分配的用于ORB类型的ORB类型的ORB类型ID。 任何人都可以通过向OMG提交ORB类型的简短（一段）描述来注册任何ORB类型，并将收到一个新的ORB类型ID作为回报。 将在OMG Web服务器上提供ORB类型说明和值的列表。

        TAG_ORB_TYPE组件在任何IOR配置文件中最多可以出现一次。 对于支持IIOP 1.1或更高版本的型材，可选地存在。
