        static final int value
        桩，关系，价值观和价值助手支持类下载。 该规范允许将代码库信息传输到存根和关系的电线上，并且在相关时可以使用预先存在的ClassLoaders。

        对于值和值助手，代码库将在值标签之后传输。 对于存根和联系，代码库作为TaggedComponent传输TAG_JAVA_CODEBASE在IOR配置文件，其中component_data是写成一个IDL字符串的代码库的CDR封装。 代码库是一个空格分隔的一个或多个URL列表。
