    public interface ObjectReferenceTemplate

    对象引用模板。 对于在ORB中创建的每个对象适配器，必须存在此实例。 server_id，orb_id和adapter_name属性在IMR的范围内唯一标识此模板。 请注意，adapter_id在同一范围内类似地是唯一的，但它是不透明的，在许多情况下不太有用。
