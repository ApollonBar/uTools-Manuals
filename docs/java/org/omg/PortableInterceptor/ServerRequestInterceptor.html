    服务器端请求拦截器。

    请求拦截器旨在通过特定点的ORB截取请求/回复序列的流程，以便服务可以查询请求信息并操纵客户端和服务器之间传播的服务上下文。 请求拦截器的主要用途是使ORB服务能够在客户端和服务器之间传输上下文信息。 请求拦截器有两种类型：客户端和服务器端。

    要编写服务器端Interceptor，请实现ServerRequestInterceptor接口。
