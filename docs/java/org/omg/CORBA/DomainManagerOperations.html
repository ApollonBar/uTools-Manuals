    DomainManager 

    public interface DomainManagerOperations
    提供DomainManager访问策略的方法。

    DomainManager已经将其与特定域的策略对象相关联。 域管理员还记录域的成员资格，并提供添加和删除成员的方法。 域管理器本身是域的成员，可能是其管理的域。 域管理器提供用于建立和导航与上级和下级域的关系以及创建和访问策略的机制。
