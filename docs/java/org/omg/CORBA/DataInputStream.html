org.omg.CORBA

Interface DataInputStream

  • All Superinterfaces:
    IDLEntity ， Serializable ， ValueBase 

    public interface DataInputStream
extends ValueBase
    定义用于从输入流读取原始数据类型的方法，以解除对自定义值类型的调整。 此界面由用户编写的自定义解组编码用于自定义值类型。
    另请参见：
    DataOutputStream ， CustomMarshal
