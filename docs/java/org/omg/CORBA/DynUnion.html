    @Deprecated
public interface DynUnion
extends Object, DynAny
    DynUnion接口表示与IDL联合关联的DynAny对象。 可以使用DynAny中定义的操作DynAny联合值。 联盟中的第一个组件对应于鉴别器; 第二个对应于联盟的实际值。 两次调用方法next()允许您访问这两个组件。
