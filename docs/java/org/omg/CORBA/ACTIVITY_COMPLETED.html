    public final class ACTIVITY_COMPLETED
extends SystemException
    可以针对其访问Activity上下文的任何方法引发ACTIVITY_COMPLETED系统异常。 它表示由于活动本身或包含活动的事务超时或活动以不同于原始请求的方式完成，因此进行方法调用的活动上下文已经完成。
    从以下版本开始：
    J2SE 1.5
