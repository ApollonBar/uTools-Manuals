    • 方法详细信息

      • policy_type

        int policy_type()
        返回与策略对象类型对应的常量值。 策略对象的值由OMG分配。 PolicyType的新值应通过发送邮件到request@omg.org从OMG获得。 一般来说，分配的常量值与相应策略对象的定义一起定义。
        结果
        对应于策略对象类型的常量值

      • copy

        Policy copy()
        返回Policy对象的副本。 该副本不保留该策略与任何域或对象的任何关系。
        结果
        Policy对象的副本

      • destroy

        void destroy()
        销毁Policy对象。 Policy对象的责任是确定是否可以被销毁。
