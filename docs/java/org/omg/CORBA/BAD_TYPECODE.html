    public final class BAD_TYPECODE
extends SystemException
    当ORB遇到格式错误的代码时（例如，具有无效TCKind值的类型代码），抛出异常。

    它包含一个次要代码，它提供了有关导致异常的更多详细信息和完成状态。 它也可以包含描述异常的字符串。

    从以下版本开始：
    JDK1.2
