Interface VM_TRUNCATABLE


  • public interface VM_TRUNCATABLE
    定义用于在类型代码中表示可截断值类型的代码。 如果值类型从另一个值类型继承“安全”，则值类型是可截断的，这意味着它可以转换为更一般的继承类型。 这是在type_modifier上的TypeCode方法的可能结果之一。
