Interface ValueFactory


  • public interface ValueFactory
    ValueFactory接口是IDL类型CORBA :: ValueFactory的本机映射。 在解析值类型的过程中，read_value（）方法由ORB运行时调用。 用户应实施此方法作为实现类型特定值工厂的一部分。 在实现中，用户将使用未初始化的价值类型调用is.read_value（java.io.Serializable）来用于解组。 流返回的值与传入的值相同，所有数据都被解组。
