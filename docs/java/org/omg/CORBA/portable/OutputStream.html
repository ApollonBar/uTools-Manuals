    public abstract class OutputStream
extends OutputStream
    OuputStream是用于将IDL类型写入CDR元数据流的Java API。 ORB使用这些方法来组织IDL类型以及将IDL类型插入到Anys中。 _array版本的方法可以直接用于编写IDL类型的序列和数组。
    从以下版本开始：
    JDK1.2
