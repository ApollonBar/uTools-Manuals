    public interface ValueBase

    对应于类型IDL类型的生成的Java类实现此接口。 换句话说，值类对象的Java映射实现了ValueBase接口。 生成的值类型的Java类应为相应的值类型提供ValueBase接口的实现。 对于可流式（即非定制）的值类型，生成的Java类还将为org.omg.CORBA.portable.Streamable接口提供一个实现。 （CORBA :: ValueBase映射到java.io.Serializable。）
