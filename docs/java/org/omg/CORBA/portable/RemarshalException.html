    public final class RemarshalException
extends 异常
    该类用于将向前的异常和对象前向GIOP消息定位回ORB。 在这种情况下，ORB必须再次请求该请求。 使用基于流的模型的存根将捕获从RemarshalException的_invoke()方法可能抛出的ObjectImpl 。 捕获异常后，存根将立即通过调用_request()来_request()请求，编组参数（如果有的话），然后调用_invoke() 。 存根应重复此过程，直到_invoke()正常返回或引起一些例外，除RemarshalException 。
