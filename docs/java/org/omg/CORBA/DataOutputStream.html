org.omg.CORBA

Interface DataOutputStream

  • All Superinterfaces:
    IDLEntity ， Serializable ， ValueBase 

    public interface DataOutputStream
extends ValueBase
    定义用于将原始数据类型写入到编组自定义值类型的输出流中的方法。 此界面由用户编写的自定义编组代码用于自定义值类型。
    另请参见：
    DataInputStream ， CustomMarshal
