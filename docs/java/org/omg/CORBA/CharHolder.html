    持有人为Char 。 有关Holder文件的更多信息，请参阅"Generated Files: Holder Files" 。

    要的Holder类， char ，其用于存储“out”和“INOUT” IDL方法参数。 如果IDL方法签名将IDL char作为“out”或“INOUT”参数，则程序员必须传递的一个实例CharHolder如在方法调用对应的参数; 对于“inout”参数，程序员还必须填写要发送到服务器的“in”值。 在方法调用返回之前，ORB将填写与从服务器返回的“out”值对应的值。

    如果myCharHolder是实例CharHolder ，存储在其值value字段可以与被访问myCharHolder.value 。

    从以下版本开始：
    JDK1.2
