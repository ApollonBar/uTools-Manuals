Interface CustomMarshal

  • All Known Subinterfaces:
    CustomValue 

    public interface CustomMarshal
    意图由ORB使用的抽象值类型，而不是用户。 在语义上，它被视为一个自定义值类型的隐式基类，尽管自定义的值类型并不会在IDL中实际继承。 自定义值类型的实现者应提供CustomMarshal 操作的实现。 这样做的方式在IDL到Java langauge映射中指定。 每个自定义的封送值类型应具有自己的实现。
