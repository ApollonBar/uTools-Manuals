    public final class INV_FLAG
extends SystemException
    当一个无效标志传递给一个操作时（例如创建一个DII请求）时抛出异常。

    它包含一个次要代码，它提供了有关导致异常的更多详细信息和完成状态。 它也可以包含描述异常的字符串。

    从以下版本开始：
    JDK1.2
