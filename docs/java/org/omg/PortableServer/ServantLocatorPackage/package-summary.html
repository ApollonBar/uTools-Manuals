Package org.omg.PortableServer.ServantLocatorPackage Description

提供用于定位仆人的类和接口。

该包提供CookieHolder类，用于传递Cookie类型作为out参数。 CookieHolder类与基本类型的其他持有CookieHolder完全相同。
