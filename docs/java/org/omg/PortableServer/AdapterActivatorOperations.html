    • 方法详细信息

      • unknown_adapter

        boolean unknown_adapter(POA parent,
                        String name)
        当ORB接收到用于标识不存在的目标POA的对象引用的请求时，调用此操作。 ORB将为必须创建的每个POA调用此操作一次，以使目标POA存在。
        参数
        parent - 表示需要创建的POA的父POA。
        name - 识别相对于父母的POA的名称。
        结果
        returns TRUE if the POA was created or FALSE otherwise.
