Package org.omg.PortableServer.CurrentPackage Description

提供方法实现，可以访问调用该方法的对象的标识。 Current包支持实现多个对象的仆人，但可以在任何仆人的POA调度方法调用的上下文中使用。 为了提供位置透明性，需要的ORB支持使用Current在本地和远程调用操作的上下文。 Current的实例可以通过发出CORBA::ORB::resolve_initial_references("POACurrent")获得。 此后，它可以在由POA分派的方法的上下文中使用，以获得标识该操作被调用的对象的POA和ObjectId 。
