Package org.omg.PortableServer.POAPackage Description

允许程序员构建可在不同ORB产品之间移植的对象实现。

便携式对象适配器（POA）旨在满足以下目标：

  • 允许程序员构建可在不同ORB产品之间移植的对象实现。
  • 为具有持久身份的对象提供支持。
  • 提供对透明激活对象的支持。
  • 允许单个仆人同时支持多个对象标识。

鉴于My.idl定义的接口My，文件MyPOA.java由idlj编译器产生。 您必须提供My的实现 ，它必须从MyPOA继承。

MyPOA.java是一个基于流的骨架，扩展了org.omg.PortableServer.Servant ，并实现了InvokeHandler接口和与骨架实现的IDL接口相关联的操作界面。

便携式对象适配器（POA）的PortableServer模块定义了本机Servant类型。 在Java编程语言中， Servant类型映射到Java org.omg.PortableServer.Servant类。 它用作所有POA服务器实现的基类，并提供了可以由应用程序员调用的多种方法，以及由POA自身调用并可被用户覆盖以控制仆人行为方面的方法。

所有映射对应于OMG文件格式/ 99-10-07.pdf指定的CORBA V2.3.1第11章。 例外是id属性，它在ptc / 00-08-06中添加，第11.3.8.26节。
