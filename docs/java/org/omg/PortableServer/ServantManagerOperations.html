    public interface ServantManagerOperations
    当POA接收到针对非活动对象的请求时，仆人经理为POA提供激活对象的能力。 仆人经理以POA作为回调对象注册，必要时由POA调用。 服务管理员可以是ServantActivators或ServantLocators。 ServantManager对象必须是包含其注册的POA对象的进程的本地。
