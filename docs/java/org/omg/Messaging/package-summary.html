Package org.omg.Messaging Description

这个软件包包含在OMG CORBA消息说明书中，所指定的Messaging模块http://cgi.omg.org/cgi-bin/doc?formal/99-10-07 。 有关详细信息，请参阅OMG规范。

请注意，我们不提供规范中Messaging模块的所有部分。 我们只提供在公共API中引用为返回值或参数类型的部分，最值得注意的是便携式拦截器。
