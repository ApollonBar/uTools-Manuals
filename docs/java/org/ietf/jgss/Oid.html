  • public class Oid
extends Object
    该类表示通用对象标识符（Oid）及其相关联的操作。

    OID是在GSS-API框架内用于识别机制和名称格式的分层全局可解释标识符。

    OIIS的结构和编码在ISOIEC-8824和ISOIEC-8825中定义。 例如，Kerberos V5机制的Oid表示为“1.2.840.113554.1.2.2”

    GSSName名称类包含表示GSS-API中定义的标准名称类型的公共静态Oid对象。
