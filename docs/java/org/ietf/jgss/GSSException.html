    public class GSSException
extends 异常
    发生GSS-API错误时会抛出此异常，包括任何机制特定的错误。 它可能包含主要和次要GSS-API状态代码。 主要错误代码是在此类中GSS-API级别定义的错误代码。 次要错误代码是可以提供附加信息的机制特定错误代码。 底层机制实现负责在抛出此异常时设置适当的次要状态代码。 除了将数字错误代码传递给调用者外，此类还可以从数字值到文本表示形式进行映射。

        minorString - 机构错误代码的文字说明。

        返回导致引发此异常的问题的GSS-API级别主要错误代码。 主要错误代码在此类中的机制独立GSS-API级别中定义。 可能提供更多信息的机制特定错误代码被设置为次要错误代码。
