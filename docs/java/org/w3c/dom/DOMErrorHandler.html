Interface DOMErrorHandler


  • public interface DOMErrorHandler
    DOMErrorHandler是在报告处理XML数据时发生的错误或进行其他处理（例如验证文档）时，DOM实现可以调用的回调接口。 甲DOMErrorHandler对象可以连接到一个Document使用的“错误处理” DOMConfiguration接口。 如果在操作期间需要报告多个错误，则传递给错误处理程序的错误的序列和数字取决于实现。

    使用DOM实现的应用程序有望实现此接口。

    另见Document Object Model (DOM) Level 3 Core Specification 。

        boolean handleError(DOMError error)
        当发生错误时，该错误处理程序将调用此方法。
        如果从该方法抛出异常，则被认为等同于返回true 。
        参数
        error - 描述错误的错误对象。 该对象可以通过跨多次调用handleError方法的DOM实现来重用。
        结果
        如果handleError方法返回false ，则DOM实现应尽可能停止当前处理。 如果该方法返回true ，处理可能会依照DOMError.severity继续。
