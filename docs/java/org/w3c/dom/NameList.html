Interface NameList


  • public interface NameList
    NameList接口提供了并行对的名称和命名空间值（可以是空值）的有序集合的抽象，而不定义或约束如何实现此集合。 NameList中的项目可以通过整数索引访问，从0开始。

    另见Document Object Model (DOM) Level 3 Core Specification 。

    从以下版本开始：
    DOM 3级
