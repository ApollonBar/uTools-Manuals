Interface NodeList

  • 所有已知实现类：
    IIOMetadataNode 

    public interface NodeList
    NodeList接口提供了节点的有序集合的抽象，而不定义或约束如何实现该集合。 NodeList DOM中的对象是活的。

    NodeList中的项目可以通过整数索引访问，从0开始。

    另见Document Object Model (DOM) Level 3 Core Specification 。
