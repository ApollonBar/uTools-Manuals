      • SEVERITY_WARNING

        static final short SEVERITY_WARNING
        DOMError描述的错误的DOMError是警告。 A SEVERITY_WARNING不会导致处理停止，除非DOMErrorHandler.handleError()返回false 。

        Constant Field Values

      • SEVERITY_ERROR

        static final short SEVERITY_ERROR
        DOMError的错误的DOMError是错误的。 A SEVERITY_ERROR可能不会导致处理停止，如果错误可以恢复，除非DOMErrorHandler.handleError()返回false 。
        另请参见：
        Constant Field Values

      • SEVERITY_FATAL_ERROR

        static final short SEVERITY_FATAL_ERROR
        由DOMError描述的错误的DOMError是致命的错误。 A SEVERITY_FATAL_ERROR将导致正常处理停止。 DOMErrorHandler.handleError()的返回值将被忽略，除非实现选择继续，在这种情况下行为将变得未定义。
        另请参见：
        Constant Field Values

        String getType()
        A DOMString指出在relatedData中预期哪些相关数据。 用户应参考错误的规范，以查找其DOMString类型和relatedData定义（如果有的话）。

        注意：例如，当“split-cdata-sections”参数正在使用时， Document.normalizeDocument()会生成警告。 因此，该方法产生一个SEVERITY_WARNING与type "cdata-sections-splitted"和第一CDATASection节点按文档顺序从分割所得的由返回relatedData属性。
