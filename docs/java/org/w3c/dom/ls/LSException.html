    Serializable 

    public class LSException
extends RuntimeException
    如果处理停止，解析器或写入操作可能会抛出LSException 。 该处理可由于停下来一DOMError用的严重性DOMError.SEVERITY_FATAL_ERROR或非恢复DOMError.SEVERITY_ERROR ，或DOMErrorHandler.handleError()返回false 。

    注意：如DOMError界面中常量的DOMError所示，DOM实现可能会在致命错误后选择继续，但是所得到的DOM树就是依赖于实现。

    另见Document Object Model (DOM) Level 3 Load and Save Specification 。
