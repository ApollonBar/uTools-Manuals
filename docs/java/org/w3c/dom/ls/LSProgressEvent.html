Interface LSProgressEvent

  • All Superinterfaces:
    Event 

    public interface LSProgressEvent
extends Event
    此接口表示一个进度事件对象，该对象在解析文档时通知应用程序进度。 它扩展了Event在[定义的接口DOM Level 3 Events ]。

    用于属性position和totalSize单位未指定，并且可以依赖于实现和输入。

    另见Document Object Model (DOM) Level 3 Load and Save Specification 。
