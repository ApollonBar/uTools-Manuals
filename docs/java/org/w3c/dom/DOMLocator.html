Interface DOMLocator


  • public interface DOMLocator
    DOMLocator是描述位置的接口（例如发生错误的地方）。

    另见Document Object Model (DOM) Level 3 Core Specification 。

    从以下版本开始：
    DOM 3级

      列号此定位器指向，或 -1如果没有列数可用。
