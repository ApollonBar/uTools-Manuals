      • initUIEvent

        void initUIEvent(String typeArg,
                 boolean canBubbleArg,
                 boolean cancelableArg,
                 AbstractView viewArg,
                 int detailArg)
        initUIEvent方法用于UIEvent通过UIEvent创建的DocumentEvent值。 在之前，此方法只能被称为UIEvent已经通过被派遣dispatchEvent方法，虽然它可能在该阶段，如果需要多次调用。 如果多次调用，则最终调用优先。
        参数
        typeArg - 指定事件类型。
        canBubbleArg - 指定事件是否可以起泡。
        cancelableArg - 指定是否可以防止事件的默认动作。
        viewArg - 指定 Event的 AbstractView 。
        detailArg - 指定 Event的详细信息。
