    public class IllegalAccessException
extends ReflectiveOperationException
    当应用程序尝试反射创建实例（数组除外），设置或获取字段或调用方法时，会抛出IllegalAccessException，但是当前执行的方法无法访问指定类，字段的定义，方法或构造函数。
