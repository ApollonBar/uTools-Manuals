      • read

        int read(CharBuffer cb)
  throws IOException
        尝试将字符读入指定的字符缓冲区。 缓冲区用作字符存储库：唯一的更改是put操作的结果。 不执行缓冲器的翻转或倒带。
        参数
        cb - 读取字符的缓冲区
        结果
        的数量 char添加到缓冲器值，或者-1，如果字符的这个源是在其端部
        异常
        IOException - 如果发生I / O错误
        NullPointerException - 如果cb为空
        ReadOnlyBufferException - 如果cb是只读缓冲区
