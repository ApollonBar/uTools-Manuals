        Type getGenericComponentType()
        返回表示此数组的组件类型的Type对象。 此方法创建数组的组件类型。 见的声明ParameterizedType为参数化类型创建过程的语义，看看TypeVariable为类型变量的创建过程。
        结果
        一个 Type对象，表示该数组的组件类型
        异常
        TypeNotPresentException - 如果底层数组类型的组件类型引用不存在的类型声明
        MalformedParameterizedTypeException - 如果底层数组类型的组件类型是指由于任何原因无法实例化的参数化类型
