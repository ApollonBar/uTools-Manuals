    public class IncompleteAnnotationException
extends RuntimeException
    抛出以指示程序已尝试访问在编辑（或序列化）注释后添加到注释类型定义的注释类型的元素。 如果新元素具有默认值，则不会抛出此异常。 API used to read annotations reflectively可以抛出这个异常 。
