Package java.lang.annotation

为Java编程语言注释工具提供库支持。

See: 描述

  • 接口摘要  
    接口 描述
    Annotation
    所有注释类型扩展的公共接口。
  • 枚举摘要  
    Enum 描述
    ElementType
    这个枚举类型的常量提供了在Java程序中可能出现注释的句法位置的简单分类。
    RetentionPolicy
    注释保留政策。
  • 异常摘要  
    异常 描述
    AnnotationTypeMismatchException
    抛出以指示程序已尝试访问注释编辑（或序列化）后类型已更改的注释元素。
    IncompleteAnnotationException
    抛出以指示程序已尝试访问在编辑（或序列化）注释后添加到注释类型定义的注释类型的元素。
  • Error Summary  
    Error 描述
    AnnotationFormatError
    当注释解析器尝试从类文件读取注释并确定注释格式错误时抛出。
  • 注解类型摘要  
    Annotation Type 描述
    Documented
    表示具有类型的注释默认情况下由javadoc和类似工具记录。
    Inherited
    表示注释类型自动继承。
    Native
    表示定义常量值的字段可以从本地代码引用。
    Repeatable
    注释类型 java.lang.annotation.Repeatable用于表示其（meta-）注释声明的注释类型是可 重复的 。
    Retention
    指示要注释具有注释类型的注释的保留时间。
    Target
    指示注释类型适用的上下文。
