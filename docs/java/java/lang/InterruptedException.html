    public class InterruptedException
extends 异常
    线程在等待，睡眠或以其他方式占用时抛出，线程在活动之前或活动期间中断。 偶尔，一个方法可能希望测试当前线程是否已被中断，如果是，立即抛出该异常。 可以使用以下代码来实现这一效果： 
      if (Thread.interrupted())  // Clears interrupted status!
      throw new InterruptedException();
