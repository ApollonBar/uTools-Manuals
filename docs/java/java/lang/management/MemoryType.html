        public static final MemoryType NON_HEAP
        非堆内存类型。

        Java虚拟机管理堆以外的内存（称为非堆内存 ）。 非堆内存包括Java虚拟机的内部处理或优化所需的方法区域和内存。 它存储每个类的结构，例如运行时常量池，字段和方法数据，以及方法和构造函数的代码。
