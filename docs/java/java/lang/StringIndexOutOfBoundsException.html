    public class StringIndexOutOfBoundsException
extends IndexOutOfBoundsException
    抛出String方法来指示索引为负或大于字符串的大小。 对于某些方法（如charAt方法），当索引等于字符串的大小时，也会抛出此异常。
    从以下版本开始：
    JDK1.0
    另请参见：
    String.charAt(int) ， Serialized Form
