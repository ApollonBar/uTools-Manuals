Class InheritableThreadLocal<T>

  • java.lang.Object

  • public class InheritableThreadLocal<T>
extends ThreadLocal<T>
    该类扩展了ThreadLocal以提供从父线程到子线程的值的继承：当子线程被创建时，子接收到父值具有值的所有可继承的线程局部变量的初始值。 孩子的价值通常与父母的价值相同; 然而，通过覆盖本课程中的childValue方法，孩子的价值可以成为父母的任意功能。

    当维护变量中的每个线程属性（例如，用户ID，事务ID）必须自动发送到创建的任何子线程时，可以使用可继承的线程局部变量。
