        返回一个新的NumberFormat实例，它为指定的区域设置格式化整数值。 返回的数字格式配置为使用半偶数舍入（参见HALF_EVEN ）将浮点数舍入到最接近的整数，以进行格式化，并仅解析输入字符串的整数部分（参见isParseIntegerOnly ）。
        参数
        locale - 所需的语言环境
        结果
        整数值的数字格式
