    public static class DateFormat.Field
extends Format.Field
    定义在AttributedCharacterIterator返回的DateFormat.formatToCharacterIterator中用作属性键的AttributedCharacterIterator ，以及FieldPosition中的字段FieldPosition 。

    该类还提供了两种方法来映射其常量和相应的日历常量。

    1.4
