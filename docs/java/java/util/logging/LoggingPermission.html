    public final class LoggingPermission
extends BasicPermission
    当SecurityManager运行的代码调用其中一个日志控件方法（如Logger.setLevel）时，SecurityManager将检查该权限。

    目前只有一个名为LoggingPermission。 这是“控制”，它允许控制日志记录配置的能力，例如添加或删除处理程序，添加或删除过滤器或更改日志记录级别。

    程序员通常不会直接创建LoggingPermission对象。 相反，它们是基于阅读安全策略文件由安全策略代码创建的。
