  • public class ConsoleHandler
extends StreamHandler
    这Handler将日志记录发布到System.err 。 默认情况下， SimpleFormatter用于生成简要摘要。

    配置：默认情况下，每个ConsoleHandler都使用以下LogManager 配置属性进行初始化，其中<handler-name>引用处理程序的完全限定类名。 如果未定义属性（或具有无效值），则使用指定的默认值。

    • <处理程序名> .LEVEL指定了Handler（默认为Level.INFO）的默认级别。
    • <handler-name> .filter指定要使用的Filter类的名称（默认为Filter ）。
    • <handler-name> .formatter指定要使用的Formatter类的名称（默认为java.util.logging.SimpleFormatter ）。
    • <handler-name>。编码要使用的字符集编码的名称（默认为默认平台编码）。

    例如，ConsoleHandler的ConsoleHandler将是：

    • java.util.logging.ConsoleHandler.level = INFO
    • java.util.logging.ConsoleHandler.formatter = java.util.logging.SimpleFormatter

    对于自定义处理程序，例如com.foo.MyHandler，属性将是：

    • com.foo.MyHandler.level = INFO
    • com.foo.MyHandler.formatter = java.util.logging.SimpleFormatter
