  • public class XMLFormatter
extends Formatter
    将LogRecord格式化为标准XML格式。

    DTD规范作为Java Logging API规范的附录A提供。

    XMLFormatter可以使用任意字符编码，但建议通常与UTF-8一起使用。 字符编码可以在输出处理程序中设置。
