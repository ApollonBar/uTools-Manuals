Class ErrorManager

  • java.lang.Object
    • java.util.logging.ErrorManager

  • public class ErrorManager
extends Object
    ErrorManager对象可以附加到处理程序来处理在记录期间在处理程序中发生的任何错误。

    处理日志记录输出时，如果处理程序遇到问题，而不是将异常返回给日志调用的发行者（谁不太可能感兴趣），则处理程序应调用其关联的ErrorManager。
