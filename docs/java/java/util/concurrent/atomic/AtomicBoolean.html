    public class AtomicBoolean

    一个boolean值可以用原子更新。 有关原子变量属性的描述，请参阅java.util.concurrent.atomic包规范。 一个AtomicBoolean用于诸如原子更新标志的应用程序，不能用作替代Boolean 。
