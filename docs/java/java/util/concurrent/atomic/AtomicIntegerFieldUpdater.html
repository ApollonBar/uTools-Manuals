Class AtomicIntegerFieldUpdater<T>

  • java.lang.Object
    • java.util.concurrent.atomic.AtomicIntegerFieldUpdater<T>
  • 参数类型
    T - 持有可更新字段的对象的类型 

    public abstract class AtomicIntegerFieldUpdater<T>
extends Object
    基于反射的实用程序，可以对指定类的指定的volatile int字段进行原子更新。 该类设计用于原子数据结构，其中同一节点的多个字段独立受原子更新的影响。

    请注意，该类中compareAndSet方法的保证弱于其他原子类。 因为这个类不能确保该字段的所有使用都适用于原子访问的目的，所以它可以保证原子性仅在相同更新器上的compareAndSet和set其他调用。
