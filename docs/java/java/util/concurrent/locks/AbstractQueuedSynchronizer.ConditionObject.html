      • await

        public final void await()
                 throws InterruptedException
        实现可中断条件等待。
        1. 如果当前线程中断，则抛出InterruptedException。
        2. 保存由AbstractQueuedSynchronizer.getState()返回的锁定状态。
        3. 调用AbstractQueuedSynchronizer.release(int)保存状态作为参数，如果失败，则抛出IllegalMonitorStateException。
        4. 阻塞直到发信号或中断。
        5. 通过调用具有保存状态的特定版本的AbstractQueuedSynchronizer.acquire(int)作为参数来重新获取。
        6. 如果在步骤4中被阻塞时中断，则抛出InterruptedException。
        Specified by:
