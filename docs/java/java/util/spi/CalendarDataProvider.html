        public abstract int getFirstDayOfWeek(Locale locale)
        在给定的locale中返回一周的第一天。 此信息由需要Calendar支持在一周有关日历字段的操作。
        参数
        locale - 所需的语言环境
        结果
        一个星期的第一天; 一个Calendar.SUNDAY .. Calendar.SATURDAY ，或0如果该值不可用于locale
        异常
        NullPointerException - 如果 locale是 null 。

        public abstract int getMinimalDaysInFirstWeek(Locale locale)
        返回一年中第一周所需的最少天数。 Calendar要求这一信息来确定一年的第一周。 请参阅how Calendar determines the first week的说明 。
        参数
        locale - 所需的语言环境
        结果
        第一周的最短天数，如果该值不适用于 locale
        异常
        NullPointerException - 如果 locale是 null 。
