    public class Attributes
extends Object
implements Map<Object,Object>, Cloneable
    属性类将Manifest属性名称映射到关联的字符串值。 有效的属性名称不区分大小写，限于set [0-9a-zA-Z_-]中的ASCII字符，长度不能超过70个字符。 属性值可以包含任何字符，并且在写入输出流时将被UTF8编码。 有关有效的属性名称和值的更多信息，请参阅JAR File Specification 。
