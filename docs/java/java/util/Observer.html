      • update

        void update(Observable o,
            Object arg)
        每当观察到的对象发生变化时，都会调用此方法。 一个应用程序调用一个Observable对象的notifyObservers方法，让所有对象的观察者通知更改。
        参数
        o - 可观察对象。
        arg - an argument passed to the notifyObservers method.
