Package java.util.regex Description

匹配字符序列与正则表达式指定的模式的类。

Pattern类的一个实例表示一个正则表达式，它以类似于Perl使用的语法以字符串形式指定。

所述的实例Matcher类被用来对一个给定的模式相匹配的字符序列。 输入通过CharSequence接口提供给匹配器，以支持与来自各种输入源的字符匹配。
