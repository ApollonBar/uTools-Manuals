Package java.util.prefs Description

该软件包允许应用程序存储和检索用户和系统首选项和配置数据。 该数据永久存储在实现相关的后备存储中。 有两个单独的首选节点树，一个用于用户首选项，一个用于系统首选项。
从以下版本开始：
JDK1.4
