Interface PreferencesFactory


  • public interface PreferencesFactory
    生成首选项对象的工厂对象。 新的Preferences实现的提供商应提供相应的PreferencesFactory实现，以便可以安装新的Preferences实现来代替平台特定的默认实现。

    这个类只适用于Preferences的实现者。 Preferences工厂的正常用户不需要参考本文档。

    从以下版本开始：
    1.4

      • userRoot

        Preferences userRoot()
        返回与主叫用户对应的用户根优先级节点。 在服务器中，返回的值通常取决于一些隐式的客户端上下文。
        结果
        用户根优先级节点对应于主叫用户
