    public class NodeChangeEvent

    Preferences节点发出的指示该节点的子节点被添加或删除的事件。

    请注意，尽管NodeChangeEvent继承了java.util.EventObject中的Serializable接口，但它并不是可串行化的。 实现适当的序列化方法来抛出NotSerializableException。
