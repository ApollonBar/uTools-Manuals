    Serializable 

    public class PreferenceChangeEvent
extends EventObject
    由Preferences节点发出的一个事件，表明已经添加，删除了某个偏好，或者改变了其值。

    请注意，尽管PreferenceChangeEvent从EventObject继承了Serializable接口，但它并不是可串行化的。 实现适当的序列化方法来抛出NotSerializableException。

    从以下版本开始：
    1.4
