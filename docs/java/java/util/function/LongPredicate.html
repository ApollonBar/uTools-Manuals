Interface LongPredicate

  • Functional Interface:
    这是一个功能界面，因此可以用作lambda表达式或方法引用的赋值对象。 

    @FunctionalInterface
public interface LongPredicate
    表示一个long值参数的谓词（布尔值函数）。 这是long消耗原始类型专业化的Predicate 。

    这是一个functional interface的功能方法是test(long) 。

    从以下版本开始：
    1.8

    • 方法详细信息

      • test

        boolean test(long value)
        在给定的参数上评估这个谓词。
        参数
        value - 输入参数
        结果
        true如果输入参数与谓词匹配，否则为 false

      • and

        default LongPredicate and(LongPredicate other)
        返回一个组合的谓词，表示该谓词与另一个谓词的短路逻辑AND。 评估组合谓词时，如果此谓词为false ，则不other other谓词。

        在评估任一谓词期间抛出的任何异常被中继到调用者; 如果此断言的评价抛出一个异常， other断言不会被评估。

        参数
        other - 将与此谓词进行逻辑AND的谓词
        结果
        一个代表该谓词和other谓词的短路逻辑AND的 other谓词
        异常
        NullPointerException - 如果其他为空

      • negate

        default LongPredicate negate()
        返回表示此谓词的逻辑否定的谓词。
        结果
        一个表示该谓词的逻辑否定的谓词

      • or

        default LongPredicate or(LongPredicate other)
        返回一个组合的谓词，表示该谓词与另一个谓词的短路逻辑或。 评估组合谓词时，如果此谓词为true ，则不other other谓词。

        在评估任一谓词期间抛出的任何异常被中继到调用者; 如果此断言的评价抛出一个异常， other断言不会被评估。

        参数
        other - 与该谓词进行逻辑关系的谓词
        结果
        一个代表该谓词与other谓词的短路逻辑或的 other谓词
        异常
        NullPointerException - 如果其他为null
