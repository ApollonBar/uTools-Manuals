Interface ObjLongConsumer<T>

  • 参数类型
    T - 操作的对象参数的类型
    Functional Interface:
    这是一个功能界面，因此可以用作lambda表达式或方法引用的赋值对象。 

    @FunctionalInterface
public interface ObjLongConsumer<T>
    表示接受对象值和long值参数的操作，并且不返回任何结果。 这是(reference, long)专业化的BiConsumer 。 与大多数其他功能界面不同，预计ObjLongConsumer将通过副作用进行操作。

    这是一个functional interface的功能方法是accept(Object, long) 。

    从以下版本开始：
    1.8

    BiConsumer
