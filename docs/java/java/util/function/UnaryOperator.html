    这是一个功能界面，因此可以用作lambda表达式或方法引用的赋值对象。 

    @FunctionalInterface
public interface UnaryOperator<T>
extends Function<T,T>
    表示对单个操作数产生与其操作数相同类型的结果的操作。 对于操作数和结果是相同类型的情况，这是Function的Function 。

    这是一个functional interface的功能方法是Function.apply(Object) 。
