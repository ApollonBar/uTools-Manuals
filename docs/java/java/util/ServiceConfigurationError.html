    public class ServiceConfigurationError
extends Error
    加载服务提供商时发生错误时抛出错误。

    在以下情况下将抛出此错误：

    • 提供者配置文件的格式违反了specification ;
    • 一个IOException而读一提供者配置文件时发生;
    • 在提供者配置文件中命名的具体提供程序类无法找到;
    • 具体的提供者类不是服务类的子类;
    • 一个具体的提供者类不能被实例化; 要么
    • 发生其他一些错误。
    从以下版本开始：
    1.6
