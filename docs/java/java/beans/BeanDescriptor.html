  • public class BeanDescriptor
extends FeatureDescriptor
    BeanDescriptor提供有关“bean”的全局信息，包括其Java类，其displayName等。

    这是BeanInfo对象返回的描述符之一，它也返回属性，方法和事件的描述符。
