    public class VetoableChangeListenerProxy

    它扩展了一类EventListenerProxy专门用于添加VetoableChangeListener具有“受限”属性。 此类的实例可以作为VetoableChangeListener添加到支持触发可变换事件的bean。

    如果对象具有getVetoableChangeListeners方法，则返回的数组可能是VetoableChangeListener和VetoableChangeListenerProxy对象的混合。

    1.4
