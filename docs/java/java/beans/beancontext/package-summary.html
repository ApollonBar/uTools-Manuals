Package java.beans.beancontext Description

提供与bean上下文相关的类和接口。 bean上下文是一个bean的容器，并定义了它包含的bean的执行环境。 单个bean上下文中可以有几个bean，而且一个bean上下文可以嵌套在另一个bean上下文中。 此包还包含从bean上下文中添加和删除的bean的事件和侦听器接口。
