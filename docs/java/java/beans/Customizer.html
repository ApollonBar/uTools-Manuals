java.beans

Interface Customizer


  • public interface Customizer
    定制器类提供了一个完整的自定义GUI来定制目标Java Bean。

    每个定制程序都应该继承自java.awt.Component类，以便可以在AWT对话框或面板中实例化。

    每个定制器应该有一个空构造函数。

      • addPropertyChangeListener

        void addPropertyChangeListener(PropertyChangeListener listener)
        注册PropertyChange事件的侦听器。 每当它以可能需要刷新显示的属性的方式更改目标bean时，定制程序将触发PropertyChange事件。
        参数
        listener - 触发PropertyChange事件时调用的对象。
