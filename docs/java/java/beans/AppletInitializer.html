    • 方法详细信息

      • initialize

        void initialize(Applet newAppletBean,
                BeanContext bCtxt)

        如果传递给java.beans.Beans.instantiate的相应变体，则将调用此方法，以便将新实例化的Applet（JavaBean）与其AppletContext，AppletStub和Container关联。

        一致的实施应：

        1. 将新实例化的Applet与适当的AppletContext关联。
        2. 实例化AppletStub（），并通过调用setStub（）将AppletStub与Applet相关联。
        3. 如果BeanContext参数为null，则它将通过调用add（）将Applet与其适当的容器相关联，将该Applet添加到其Container中。 如果BeanContext参数是非空值，则BeanContext有责任在随后调用其addChildren（）方法时将Applet与其Container关联。
        参数
        newAppletBean - 新实例化的JavaBean
        bCtxt - 用于此Applet的BeanContext，或null。

      • activate

        void activate(Applet newApplet)

        激活，和/或标记Applet活动。 此接口的实现者应将该Applet标记为活动状态，并可选地调用其start（）方法。

        参数
        newApplet - 新实例化的JavaBean
