    public class FeatureDescriptor
extends Object
    FeatureDescriptor类是PropertyDescriptor，EventSetDescriptor和MethodDescriptor等的常见基类。

    它支持一些可以为任何内省描述符设置和检索的常见信息。

    此外，它还提供了一个扩展机制，使得任意的属性/值对可以与设计特征相关联。
