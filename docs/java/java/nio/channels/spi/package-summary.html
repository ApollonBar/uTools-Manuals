Package java.nio.channels.spi Description

java.nio.channels包的服务提供者类。

只有定义新的选择器提供者或异步通道提供者的开发者才需要直接使用这个包。

除非另有说明，否则将null参数传递给此程序包中任何类或接口中的构造函数或方法将导致抛出NullPointerException 。
