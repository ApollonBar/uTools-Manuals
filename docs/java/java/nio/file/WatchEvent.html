Interface WatchEvent<T>

  • 参数类型
    T - 与事件关联的上下文对象的类型 

    public interface WatchEvent<T>
    的事件或为与注册的对象的重复事件WatchService 。

    事件按其kind进行分类 ，并有一个count来表示事件被观察的次数。 这样可以有效地表示重复的事件。 context方法返回与事件相关联的任何上下文。 在重复事件的情况下，所有事件的上下文是相同的。

    监视事件是不可变的，安全的可供多个并发线程使用。
