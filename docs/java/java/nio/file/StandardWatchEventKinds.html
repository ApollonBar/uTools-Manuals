      • ENTRY_CREATE

        public static final WatchEvent.Kind<Path> ENTRY_CREATE
        创建目录条目。

        当为此事件注册目录时，当观察到在目录中创建条目或重命名为目录时，WatchKey排队。 此事件的事件count始终为1 。

      • ENTRY_DELETE

        public static final WatchEvent.Kind<Path> ENTRY_DELETE
        目录条目已删除。

        当为此事件注册目录时，当观察到条目被删除或重命名为目录时， WatchKey将排队。 此事件的事件count始终为1 。

      • ENTRY_MODIFY

        public static final WatchEvent.Kind<Path> ENTRY_MODIFY
        目录条目修改。

        当为此事件注册目录时，当观察到目录中的条目已被修改时， WatchKey将排队。 此事件的count事件为1或更高。
