    public static final class AclEntry.Builder
extends Object
    AclEntry对象的构建器。

    通过调用由AclEntry类定义的newBuilder方法之一获得一个Builder对象。

    Builder对象是可变的，并且不安全地被多个并发线程使用而没有适当的同步。

    从以下版本开始：
    1.7
