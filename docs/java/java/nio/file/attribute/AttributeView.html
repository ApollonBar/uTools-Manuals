    public interface AttributeView
    提供与文件系统中的对象相关联的非不透明值的只读或可更新视图的对象。 此接口通过定义视图支持的属性的特定属性视图进行扩展或实现。 特定的属性视图通常会定义类型安全的方法来读取或更新其支持的属性。
