Interface CopyOption

  • 所有已知实现类：
    LinkOption ， StandardCopyOption 

    public interface CopyOption
    配置如何复制或移动文件的对象。

    这种类型的对象可以与使用Files.copy(Path,Path,CopyOption...) ， Files.copy(InputStream,Path,CopyOption...)和Files.move(Path,Path,CopyOption...)方法来配置一个文件是如何复制或移动。

    StandardCopyOption枚举类型定义了标准选项。
