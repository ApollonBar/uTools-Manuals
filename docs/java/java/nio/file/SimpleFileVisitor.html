        在目录中的条目之后调用目录，并且已经访问了所有的后代。

        除非被覆盖，这个方法返回CONTINUE如果目录迭代没有I / O异常完成; 否则这种方法重新抛出引起该目录的迭代的I / O异常，以过早终止。
