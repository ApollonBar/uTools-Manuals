    Serializable ， Cloneable ， Comparable < Date > 

    public class Time
extends Date

    围绕java.util.Date类的薄包装，允许JDBC API将其标识为SQL TIME值。 Time类添加了格式化和解析操作，以支持时间值的JDBC转义语法。

    日期组件应设置为1970年1月1日的“零纪元”值，不得访问。
