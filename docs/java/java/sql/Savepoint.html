Interface Savepoint


  • public interface Savepoint
    保存点的表示，它是当前事务中可以从Connection.rollback方法引用的一个点。 当事务回滚到保存点时，保存点之后所做的所有更改都将被撤销。

    保存点可以是命名的或未命名的。 未命名的保存点由基础数据源生成的ID标识。

    从以下版本开始：
    1.4
