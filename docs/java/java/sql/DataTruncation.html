    public class DataTruncation
extends SQLWarning
    由于其超出MaxFieldSize原因，当数据值意外截断时，作为DataTruncation异常（写入）或作为DataTruncation警告（读取）报告的MaxFieldSize 。

    读取期间的DataTruncation的DataTruncation为01004 。

    写入期间的DataTruncation的DataTruncation为22001 。
