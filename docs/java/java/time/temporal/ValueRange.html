    日期时间字段的有效值的范围。

    所有TemporalField实例都具有有效的值范围。 例如，ISO日期从1到28到31之间。这个类捕获有效范围。

    了解这个课程的局限性很重要。 只提供最小值和最大值。 在外部范围内可能存在无效值。 例如，奇怪的字段可能具有1,2,4,6,7的有效值，因此具有范围“1 - 7”，尽管值3和5无效。

    这个类的实例不是绑定到一个特定的领域。

    实现要求：
    这个类是不可变的和线程安全的。
