    ISO日历系统中的一个时代。

    ISO-8601标准没有定义时间。 因此，在此之前多年来，在0001-01-01（ISO）之后或之后已经有两个时代 - “当代”（CE），“当前时代”（BCE）之前的定义。

    year-of-era era proleptic-year 2 CE 2 1 CE 1 1 BCE 0 2 BCE -1

    不要使用ordinal()获得IsoEra的数字IsoEra 。 改用getValue() 。

    实现要求：
    这是一个不可变的和线程安全的枚举。
    从以下版本开始：
    1.8
