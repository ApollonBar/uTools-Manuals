    泰国佛教日历系统的时代。

    泰国佛教日历系统有两个时代。 当今时代，从1年起，被称为“佛教”时期。 所有前几年，在零食或更早的时候，在一年的时间里，这个数字是“前佛教时代”的一部分。

    year-of-era era proleptic-year ISO proleptic-year 2 BE 2 -542 1 BE 1 -543 1 BEFORE_BE 0 -544 2 BEFORE_BE -1 -545

    请勿使用ordinal()获取ThaiBuddhistEra的数字ThaiBuddhistEra 。 改用getValue() 。

    实现要求：
    这是一个不可变的和线程安全的枚举。
    从以下版本开始：
    1.8
