    public enum FormatStyle
extends Enum<FormatStyle>
    枚举本地化日期，时间或日期时间格式化的风格。

    从配置获取日期时间样式时，将使用这些样式。 参见DateTimeFormatter和DateTimeFormatterBuilder 。

    实现要求：
    这是一个不可变的和线程安全的枚举。
    从以下版本开始：
    1.8
