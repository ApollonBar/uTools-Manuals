    public enum TextStyle
extends Enum<TextStyle>
    枚举文本格式和解析的风格。

    文本样式定义格式化文本的三种尺寸 - “满”，“短”和“窄”。 这三种尺寸中的每一种均可在“标准”和“独立”变体中使用。

    在大多数语言中，三种尺寸之间的差异是显而易见的。 例如，在英语中，“满月”为“一月”，“短”月为“一月”，“窄”月为“J”。 请注意，狭窄的大小通常不是唯一的。 例如，'1月'，'6月'和'7'都有'窄'文本'J'。

    “标准”和“独立”形式之间的区别是形容词难度很大，英文没有差别。 然而，在其他语言中，单独使用文本时使用的单词有所不同，而不是完整的日期。 例如，一个月在单独使用日期选择器中使用的单词与用于与日期中的日期和年份相关联的月份的单词不同。

    实现要求：
    这是不可变的和线程安全的枚举。
