    public class RemoteException
extends IOException
    A RemoteException是在执行远程方法调用期间可能发生的许多通信相关异常的常见超类。 远程接口的每种方法，扩展java.rmi.Remote必须在其throws子句中列出RemoteException 。

    从版本1.4开始，这种异常已被改进以符合通用异常链接机制。 可以在施工时提供并通过公共detail字段访问的“包裹的远程异常”现在被称为原因 ，并且可以通过Throwable.getCause()方法以及前述的“传统字段”来访问。

    调用方法Throwable.initCause(Throwable)上的一个实例RemoteException总是抛出IllegalStateException 。

    从以下版本开始：
