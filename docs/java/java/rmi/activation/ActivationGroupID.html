    注册激活组的标识符有几个目的：
    • 在激活系统内唯一识别组，以及
    • 包含对组的激活系统的引用，以便组可以在必要时联系其激活系统。

    ActivationGroupID从呼叫返回到ActivationSystem.registerGroup ，用于标识激活系统中的组。 当创建/重新创建激活组时，此组ID将作为参数传递给激活组的特殊构造函数。

    从以下版本开始：
    1.2
