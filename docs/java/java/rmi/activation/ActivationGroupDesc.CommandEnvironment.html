      • CommandEnvironment

        public CommandEnvironment(String cmdpath,
                          String[] argv)
        创建一个具有所有必要信息的CommandEnvironment。
        参数
        cmdpath - java可执行文件的名称，包括完整路径，或null ，意思是“使用rmid的默认值”。 命名的程序必须能够接受多个-Dpropname=value选项（如“java”工具所述）
        argv - 将用于创建ActivationGroup的额外选项。 Null与空列表具有相同的效果。
        从以下版本开始：
        1.2
