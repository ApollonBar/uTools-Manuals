public class ServerRuntimeException

    从在JDK 1.1上执行的服务器执行时，抛出ServerRuntimeException作为远程方法调用的结果，当在处理服务器上的RuntimeException时抛出RuntimeException，无论是在解组参数，执行远程方法本身还是编组返回值。 一个ServerRuntimeException实例包含原始的RuntimeException作为其原因。

    ServerRuntimeException不会从在Java 2平台v1.2或更高版本上执行的服务器抛出。

    从以下版本开始：
