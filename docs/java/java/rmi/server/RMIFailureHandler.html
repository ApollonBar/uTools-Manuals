Interface RMIFailureHandler


  • public interface RMIFailureHandler
    一个RMIFailureHandler可以通过RMISocketFactory.setFailureHandler通话进行注册。 当RMI运行时无法创建ServerSocket以侦听来电时，调用处理程序的failure方法。 failure方法返回一个布尔值，指示运行时是否应尝试重新创建ServerSocket 。
    从以下版本开始：
    JDK1.1

      • failure

        boolean failure(异常 ex)
        当RMI运行时无法通过RMISocketFactory创建ServerSocket时，将调用failure回调。 一个RMIFailureHandler通过将呼叫注册RMISocketFacotry.setFailureHandler 。 如果没有安装故障处理程序，则默认行为是尝试重新创建ServerSocket。
        参数
        ex - 在 ServerSocket创建期间发生的 ServerSocket
        结果
        如果为true，RMI运行时将尝试重试 ServerSocket创建
        从以下版本开始：
        JDK1.1
        另请参见：
