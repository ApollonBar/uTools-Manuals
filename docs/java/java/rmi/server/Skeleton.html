public interface Skeleton
    Skeleton接口仅由RMI实现使用。

    由rmic stub编译器生成的每个版本1.1（和版本1.1兼容的骨架在1.2使用rmic -vcompat ）骨架类实现了这个接口。 远程对象的骨架是将调用调度到实际远程对象实现的服务器端实体。

    从以下版本开始：
    JDK1.1
