Package java.rmi.dgc Description

提供RMI分布式垃圾回收（DGC）的类和接口。 当RMI服务器将对象返回给其客户端（远程方法的调用者）时，它会跟踪远程对象在客户端中的使用情况。 当没有更多的对客户端上的远程对象的引用，或者如果引用的“租约”过期而未被更新，则服务器将垃圾收集远程对象。
从以下版本开始：
JDK1.1
