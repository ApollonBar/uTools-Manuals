Interface Group

    Principal 

    public interface Group
extends Principal
    该界面用于表示一组主体。 （主体代表一个实体，如个人用户或公司）。

    请注意，Group扩展了Principal。 因此，可以将Principal或Group作为参数传递给包含Principal参数的方法。 例如，您可以通过调用对象的addMember方法，将Principal或Group传递给Group对象，将Principal或Group添加到Group对象。

    • 方法详细信息

      • addMember

        boolean addMember(Principal user)
        将指定的成员添加到组中。
        参数
        user - 添加到此组的主体。
        结果
        如果成员成功添加，则为true，如果委托人已经是会员，则为false。

      • removeMember

        boolean removeMember(Principal user)
        从组中删除指定的成员。
        参数
        user - 从该组移除的主体。
        结果
        如果委托人被移除，则为真，如果委托人不是会员，则为假。

      • isMember

        boolean isMember(Principal member)
        如果传递的主体是组的成员，则返回true。 此方法执行递归搜索，因此如果主体属于属于此组成员的组，则返回true。
        参数
        member - 其成员将被检查的主体。
        结果
        如果主体是该组的成员，则为true，否则为false。

      • members

        Enumeration<? extends Principal> members()
        返回组中成员的枚举。 返回的对象可以是Principal或Group（它是Principal的子类）的实例。
        结果
        集团成员的列举。
