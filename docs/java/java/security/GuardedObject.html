    GuardedObject是一个用于保护对另一个对象的访问的对象。

    GuardedObject封装了一个目标对象和一个Guard对象，使得只有在Guard对象允许的情况下才能访问目标对象。 一个对象被GuardedObject封装后，对该对象的访问由getObject方法控制，该方法调用Guard对象上的守卫访问的checkGuard方法。 如果不允许访问，则抛出异常。
