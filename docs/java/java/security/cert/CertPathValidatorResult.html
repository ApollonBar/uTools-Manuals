    public interface CertPathValidatorResult
extends Cloneable
    认证路径验证器算法的结果的规范。

    该接口的目的是为所有认证路径验证器结果分组（并提供类型安全性）。 CertPathValidator.validate方法返回的所有结果都必须实现此接口。
