    public abstract class CRL
extends Object
    该类是具有不同格式但重要常见用途的证书吊销列表（CRL）的抽象。 例如，所有CRL共享列出撤销证书的功能，可以查询是否列出给定的证书。

    专门的CRL类型可以通过对这个抽象类进行子类化来定义。
