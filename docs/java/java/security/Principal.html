      • implies

        default boolean implies(Subject subject)
        如果该主体隐含指定的主题，则返回true。

        如果subject不为空，则该方法的默认实现将返回true，并且至少包含一个等于该主体的主体。

        如果需要，子类可以用不同的实现覆盖它。

        参数
        subject - Subject
        结果
        如果 subject为非空且由本校本所隐含，则为真，否则为false。
