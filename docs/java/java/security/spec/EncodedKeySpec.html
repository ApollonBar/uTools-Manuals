      • getFormat

        public abstract String getFormat()
        返回与此密钥规范相关联的编码格式的名称。

        如果密钥的不透明表示（参见Key ）可以转换（参见KeyFactory ）到此密钥规范（或其子类）中， getFormat在不透明密钥上调用的getFormat返回与该密钥规范的getFormat方法相同的值。

        结果
        编码格式的字符串表示形式。
