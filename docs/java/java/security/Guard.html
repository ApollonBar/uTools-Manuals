    public interface Guard

    该接口表示一个保护，它是一个用于保护对另一个对象的访问的对象。

    此接口包含一个单一的方法， checkGuard ，具有单个object参数。 checkGuard被调用（由GuardedObject getObject方法）来确定是否允许访问该对象。

        void checkGuard(Object object)
         throws SecurityException
        确定是否允许访问被保护对象object 。 如果允许访问，则默认返回。 否则，抛出SecurityException。
        参数
        object - 被保护者保护的对象。
        异常
