    这个类是一个输入流过滤器，它提供跟踪当前行号的附加功能。

    行是以回车符（ '\r' ），换行字符（ '\n' ）结尾的字节序列，或紧跟着换行字符的回车符。 在所有三种情况下，行终止字符都作为单个换行符返回。

    行号开始于0 ，和被增加1当read返回换行符。
