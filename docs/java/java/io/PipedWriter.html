      • connect

        public void connect(PipedReader snk)
             throws IOException
        将此管道写入器连接到接收器。 如果此对象已连接到其他管道读取器，则抛出IOException 。

        如果snk是未连接的管道读写器，并且src是未连接的管道写入器，则可以通过以下任一方式连接它们： 

         src.connect(snk)
        或电话： 
         snk.connect(src)
        两个电话有相同的效果。
        参数
        snk - 连接到管道阅读器。
        异常
        IOException - 如果发生I / O错误。
