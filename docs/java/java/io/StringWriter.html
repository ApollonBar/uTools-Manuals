  • All Implemented Interfaces:
    Closeable ， Flushable ， Appendable ， AutoCloseable 

    public class StringWriter
extends Writer
    在字符串缓冲区中收集其输出的字符流，然后可以用于构造字符串。

    关闭StringWriter没有任何效果。 在流已关闭后，可以调用此类中的方法，而不生成IOException 。

    从以下版本开始：
    JDK1.1
