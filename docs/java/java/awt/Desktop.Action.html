    public static enum Desktop.Action
extends Enum<Desktop.Action>
    表示动作类型。 每个平台支持一组不同的动作。 您可以使用Desktop.isSupported(java.awt.Desktop.Action)方法来确定当前平台是否支持给定的操作。

    1.6
    另请参见：
