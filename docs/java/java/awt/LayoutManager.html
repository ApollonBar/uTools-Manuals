    public interface LayoutManager
    定义知道如何布置Container的类的Container 。

    秋千的绘画建筑假设一个JComponent的孩子不重叠。 如果一个JComponent的LayoutManager允许孩子重叠， JComponent必须覆盖isOptimizedDrawingEnabled才能返回false。

    另请参见：
    Container ， JComponent.isOptimizedDrawingEnabled()
