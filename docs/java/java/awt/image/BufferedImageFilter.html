    public class BufferedImageFilter
extends ImageFilter
implements Cloneable
    所述BufferedImageFilter类的子类的ImageFilter是提供一种使用单一源/单一目标图像操作符（的简单方法BufferedImageOp ），用以过滤BufferedImage在图像生产者/消费者/观察员范例。 这些图像操作符的实例为： ConvolveOp ， AffineTransformOp和LookupOp 。
