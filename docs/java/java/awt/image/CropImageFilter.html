    public class CropImageFilter
extends ImageFilter
    用于裁剪图像的ImageFilter类。 该类扩展了基本的ImageFilter类，以提取现有图像的给定矩形区域，并为仅包含提取区域的新图像提供源。 它意味着与FilteredImageSource对象结合使用以产生现有图像的裁剪版本。
