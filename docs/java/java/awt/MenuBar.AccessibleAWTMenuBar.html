    protected class MenuBar.AccessibleAWTMenuBar
extends MenuComponent.AccessibleAWTMenuComponent
    MenuBar的内部类用于为辅助功能提供默认支持。 这个类不是由应用程序开发人员直接使用的，而是意味着仅由菜单组件开发人员进行子类化。

    该类实现对MenuBar类的辅助功能支持。 它提供了适用于菜单栏用户界面元素的Java Accessibility API的实现。
