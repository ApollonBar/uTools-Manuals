    public abstract class MouseDragGestureRecognizer
extends DragGestureRecognizer
implements MouseListener, MouseMotionListener
    DragGestureRecognizer这个抽象子类定义了基于鼠标的手势的DragGestureRecognizer 。 每个平台通过Toolkit.createDragGestureRecognizer（）方法实现了自己具有的这个类的具体子类，以便对启动拖放操作的依赖于平台的鼠标手势的识别进行封装。

    鼠标拖动手势识别器应该遵守拖动手势运动阈值，可通过DragSource.getDragThreshold()获得 。 只有当最新的鼠标拖动事件的位置和相应的鼠标按钮按下事件的位置之间的水平或垂直方向上的距离大于拖动手势运动阈值时才能识别拖动手势。

    使用DragSource.createDefaultDragGestureRecognizer(java.awt.Component, int, java.awt.dnd.DragGestureListener)创建的拖动手势识别器遵循此约定。
