    public class DropTarget
extends Object
implements DropTargetListener, Serializable
    该DropTarget与相关Component时Component希望接受拖放操作期间下降。

    每个DropTarget都与FlavorMap 。 默认FlavorMap以后指定FlavorMap返回的SystemFlavorMap.getDefaultFlavorMap() 。
