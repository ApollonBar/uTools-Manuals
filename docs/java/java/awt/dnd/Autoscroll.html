java.awt.dnd

Interface Autoscroll


  • public interface Autoscroll
    在DnD操作期间，用户可能希望将操作的主题放在用户当前不可见的可滚动GUI控件的区域上。

    在这种情况下，希望GUI控件检测到这一点并进行滚动操作，以使使用者可见的遮蔽区域。 此功能称为自动滚动。

    如果GUI控制既是活性DropTarget并且也可滚动的，它可以通过实现该接口通过从所述DND系统中的用户接收自动滚动手势的通知。

    的自动滚动手势是由用户通过保持拖动光标不动与所述的一个边界区域发起Component ，被称为“自动滚动区域”，为的预定时间段，这将导致在反复滚动请求到Component直到拖动Cursor恢复其动议。
