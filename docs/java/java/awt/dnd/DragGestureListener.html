Interface DragGestureListener

    public interface DragGestureListener
extends EventListener
    用于接收拖动手势事件的侦听器界面。 此界面用于拖动手势识别实现。 请参阅规范DragGestureRecognizer关于如何注册监听器接口的详细信息。 当识别拖拽手势的DragGestureRecognizer调用这个接口的dragGestureRecognized()方法并传递一个DragGestureEvent 。
    另请参见：
    DragGestureRecognizer ， DragGestureEvent ， DragSource
