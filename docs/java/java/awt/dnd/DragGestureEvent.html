    Serializable 

    public class DragGestureEvent
extends EventObject
    DragGestureEvent传递给DragGestureListener的dragGestureRecognized（）方法，当特定的DragGestureRecognizer检测到正在跟踪的Component上发生平台依赖的拖动启动手势时。 任何DragGestureEvent实例的action字段应该采用以下值之一：
    • DnDConstants.ACTION_COPY
    • DnDConstants.ACTION_MOVE
    • DnDConstants.ACTION_LINK
    分配与上述不同的值将导致未指定的行为。
    另请参见：
    DragGestureRecognizer ， DragGestureListener ， DragSource ， DnDConstants ， Serialized Form
