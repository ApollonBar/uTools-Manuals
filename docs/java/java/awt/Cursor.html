      • Cursor

        protected Cursor(String name)
        使用指定的名称创建一个新的自定义游标对象。

        注意：此构造函数只能由AWT实现使用，作为对自定义游标的支持的一部分。 应用程序应该使用Toolkit.createCustomCursor（）。

        参数
        name - 光标的用户可见名称。
        另请参见：
        Toolkit.createCustomCursor(java.awt.Image, java.awt.Point, java.lang.String)
