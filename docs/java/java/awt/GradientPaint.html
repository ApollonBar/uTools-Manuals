    public class GradientPaint
extends Object
implements Paint
    GradientPaint课程提供了一种以线性颜色渐变模式填充Shape的方法。 如果Point P1与Color C1和Point P2与Color C2在用户空间中Color ，则P1，P2连接线上的Color按比例从C1更改为C2。 扩展P1，P2连接线上的任何点P都不具有在扩展P1，P2连接线上的P的垂直投影点P'的颜色。 P1，P2段外的扩展线上的点可以以两种方式之一进行着色。
    • 如果梯度是循环的，则扩展的P1，P2连接线上的点在颜色C1和C2之间来回循环。
    • 如果梯度是非循环的，那么该段的P1侧上的点具有常数Color C1，而P2侧的点具有常数Color C2。
