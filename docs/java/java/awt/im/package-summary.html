Package java.awt.im Description

为输入法框架提供类和接口。 该包使文本编辑组件能够通过输入法接收文本输入。 输入法是允许用户以键盘上简单打字以外的方式输入文字的软件组件。 它们通常用于输入日语，中文或韩语 - 使用数千个不同的字符 - 键盘上键少得多的键盘。 然而，该框架还支持其他语言的输入方法和使用完全不同的输入机制，例如手写或语音识别。
