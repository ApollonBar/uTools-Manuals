Class PointerInfo

  • java.lang.Object
    • java.awt.PointerInfo

  • public class PointerInfo
extends Object
    一个描述指针位置的类。 它提供了GraphicsDevice其中指针是和Point表示的指针的坐标。

    该课程的实例应通过MouseInfo.getPointerInfo()获得。 当鼠标移动时， PointerInfo实例不会动态更新。 要获取更新的位置，您必须再次致电MouseInfo.getPointerInfo() 。

    从以下版本开始：
    1.5
    另请参见：
    MouseInfo.getPointerInfo()
