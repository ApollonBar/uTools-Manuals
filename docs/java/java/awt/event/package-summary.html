Package java.awt.event Description

提供用于处理AWT组件触发的不同类型事件的接口和类。 有关AWT事件模型的详细信息，请参阅java.awt.AWTEvent类。 活动由事件来源触发。 事件侦听器向事件源注册以接收关于特定类型的事件的通知。 此包定义事件和事件侦听器以及事件侦听器适配器，这些方便类使事务侦听器的写入过程更为简单。
从以下版本开始：
JDK1.1
