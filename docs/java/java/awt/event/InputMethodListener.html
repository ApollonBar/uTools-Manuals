    public interface InputMethodListener
extends EventListener
    用于接收输入法事件的侦听器接口。 文本编辑组件必须安装输入法事件侦听器才能使用输入法。

    文本编辑组件还必须提供一个InputMethodRequests的实例。

    从以下版本开始：
    1.2
