    public interface ComponentListener
extends EventListener
    用于接收组件事件的侦听器接口。 有兴趣处理组件事件的类可以实现这个接口（和它包含的所有方法），或扩展抽象的ComponentAdapter类（仅覆盖感兴趣的方法）。 然后，使用该组件的addComponentListener方法将从该类创建的侦听器对象注册到组件。 当组件的大小，位置或可见性的变化，在监听对象中的相关方法被调用，并ComponentEvent传递给它。

    组件事件仅供通知用途; AWT将自动处理组件移动并在内部调整大小，以使GUI布局正常工作，无论程序是否注册ComponentListener 。
