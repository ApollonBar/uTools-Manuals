    public abstract class HierarchyBoundsAdapter

    用于接收祖先移动和调整大小的事件的抽象适配器类。 这个类中的方法是空的。 此类存在为创建侦听器对象的方便。

    扩展此类并覆盖感兴趣的事件的方法。 （如果您实现HierarchyBoundsListener接口，则必须在其中定义两个方法，这个抽象类为它们定义了空方法，因此您只需定义所关心事件的方法。）

    使用您的类创建一个侦听器对象，然后使用组件的addHierarchyBoundsListener方法将其注册到组件。 当组件所属通过调整大小或祖先的运动改变到层次结构中，在收听者对象中的相关方法被调用，并且HierarchyEvent被传递给它。
