    一个扩展EventListenerProxy专门用于为特定事件掩码添加AWTEventListener的类。 此类的实例可以作为AWTEventListener添加到Toolkit对象。

    该getAWTEventListeners的方法Toolkit可以返回的混合物AWTEventListener个AWTEventListenerProxy对象。
