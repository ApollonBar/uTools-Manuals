    public interface TextListener
extends EventListener
    用于接收文本事件的侦听器界面。 对处理文本事件感兴趣的类实现此接口。 然后使用该类创建的对象使用组件的addTextListener方法向组件addTextListener 。 当组件的文本更改时，调用侦听器对象的textValueChanged方法。
