    public interface HierarchyListener
extends EventListener
    用于接收层次结构的侦听器界面更改事件。 有兴趣处理层次结构改变事件的类应该实现这个接口。 然后，使用组件的addHierarchyListener方法将从该类创建的侦听器对象注册到组件。 当组件所属的层次更改时，将调用侦听器对象中的hierarchyChanged方法，并将HierarchyEvent传递给它。

    层次结构事件仅供通报用途; AWT将自动处理层次结构内部的更改，以使GUI布局，可显示性和可见性正常工作，无论程序是否注册HierarchyListener 。
