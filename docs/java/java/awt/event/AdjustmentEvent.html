    Serializable 

    public class AdjustmentEvent
extends AWTEvent
    可调整对象发出的调整事件，如Scrollbar和ScrollPane 。 当用户更改滚动组件的值时，它会接收一个AdjustmentEvent的实例。

    如果未指定的行为将引起id任何特定的参数AdjustmentEvent实例不在范围从ADJUSTMENT_FIRST到ADJUSTMENT_LAST 。

    该type任何AdjustmentEvent例如采用下列值之一：

    • UNIT_INCREMENT
    • UNIT_DECREMENT
    • BLOCK_INCREMENT
    • BLOCK_DECREMENT
    • TRACK
    分配与上述不同的值将导致未指定的行为。
    从以下版本开始：
    1.1
    另请参见：
    Adjustable ， AdjustmentListener ， Serialized Form
