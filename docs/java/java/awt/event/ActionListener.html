    public interface ActionListener
extends EventListener
    用于接收动作事件的侦听器界面。 有兴趣处理动作事件的类实现此接口，并使用该组件的addActionListener方法将使用该类创建的对象注册到组件中。 当事件发生时，将调用该对象的actionPerformed方法。
    从以下版本开始：
    1.1
