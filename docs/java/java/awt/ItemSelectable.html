      • addItemListener

        void addItemListener(ItemListener l)
        当用户更改项目的状态时，添加监听器以接收项目事件。 当项目的状态以编程方式设置时，不会发送项目事件。 如果l为null ，则不抛出异常，不执行任何操作。
        参数
        l - 接收事件的侦听器
