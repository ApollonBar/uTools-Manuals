    Paper类描述了一张纸的物理特性。

    创建Paper对象时，应用程序有责任确保纸张尺寸和可成像区域兼容。 例如，如果纸张尺寸从11 x 17更改为8.5 x 11，则应用程序可能需要减少可成像区域，以便打印的任何内容符合页面。

    另请参见：
    setSize(double, double) ， setImageableArea(double, double, double, double)
