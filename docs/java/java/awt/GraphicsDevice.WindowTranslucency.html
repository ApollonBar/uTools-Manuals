    • Enum Constant Summary

      Enum Constants  
      Enum Constant and Description
      PERPIXEL_TRANSLUCENT
      表示在底层系统中支持包含或可能包含0.0和1.0之间任意α值的像素的Windows。
      PERPIXEL_TRANSPARENT
      表示对底层系统的支持，每个像素的窗口保证完全不透明，alpha值为1.0，或者完全透明，Alpha值为0.0。
