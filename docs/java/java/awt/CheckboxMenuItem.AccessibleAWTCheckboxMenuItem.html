implements AccessibleAction, AccessibleValue
    用于提供对辅助功能的默认支持的CheckboxMenuItem的内部类。 这个类不是由应用程序开发人员直接使用的，而是意味着仅由菜单组件开发人员进行子类化。

    这个类实现了可访问性支持CheckboxMenuItem类。 它提供了适用于复选框菜单项用户界面元素的Java Accessibility API的实现。
