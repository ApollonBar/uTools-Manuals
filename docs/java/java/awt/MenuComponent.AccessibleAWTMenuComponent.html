implements Serializable, AccessibleComponent, AccessibleSelection
    内部类别MenuComponent用于提供对辅助功能的默认支持。 这个类不是由应用程序开发人员直接使用的，而是意味着仅由菜单组件开发人员进行子类化。

    该类用于获取此对象的可访问角色。
