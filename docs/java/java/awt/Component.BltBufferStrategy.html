      • contentsRestored

        public boolean contentsRestored()
        描述从类别： BufferStrategy复制
        返回绘图缓冲区是否最近从丢失状态恢复并重新初始化为默认背景颜色（白色）。 由于缓冲区策略中的缓冲区通常是VolatileImage型，所以它们可能会丢失。 最近一次调用getDrawGraphics以来，如果表面最近已经从丢失状态恢复，则可能需要重新绘制。 有关丢失缓冲区的讨论，请参见VolatileImage 。
        Specified by:
        contentsRestored在类别 BufferStrategy
        结果
        绘图缓冲区是否最近从丢失状态恢复并重新初始化为默认背景颜色（白色）
