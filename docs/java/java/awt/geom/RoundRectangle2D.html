    public abstract class RoundRectangle2D
extends RectangularShape
    RoundRectangle2D类定义了一个矩形，其中包含由位置(x,y)定义的(x,y) ，尺寸为(w x h) ，以及用于围绕拐角的圆弧的宽度和高度。

    该类是存储2D圆角矩形的所有对象的抽象超类。 坐标的实际存储表示留给子类。

    从以下版本开始：
    1.2
