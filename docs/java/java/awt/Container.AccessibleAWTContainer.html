    protected class Container.AccessibleAWTContainer
extends Component.AccessibleAWTComponent
    Container的内部类用于为辅助功能提供默认支持。 这个类不是由应用程序开发人员直接使用的，而是意味着仅由容器开发人员进行子类化。

    该类用于获取此对象的可访问角色，以及在AccessibleContainer界面中实现许多方法。
