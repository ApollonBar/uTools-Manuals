    public class StringSelection
extends Object
implements Transferable, ClipboardOwner
    A Transferable ，其实现传送String所需的String 。 这Transferable正确支持DataFlavor.stringFlavor和所有相同的口味。 支持DataFlavor.plainTextFlavor和所有等效口味已被弃用 。 没有其他DataFlavor s的支持。
    另请参见：
    DataFlavor.stringFlavor ， DataFlavor.plainTextFlavor
