    public interface FlavorMap
    对应于平台特定数据格式的“本土”（Strings）和“风格”（DataFlavors）之间的双向映射，它们与平台无关的MIME类型相对应。 FlavorMaps不必是对称的，而是通常是对称的。

    1.2
