    public final class SystemFlavorMap
extends Object
implements FlavorMap, FlavorTable
    SystemFlavorMap是与“平台”特定的数据格式相对应的“本地”（Stratives）和“平台无关MIME类型”（DataFlavors）之间的可配置映射。 该映射由数据传输子系统用于在Java和本机应用程序之间以及在单独的VM中的Java应用程序之间传输数据。

    从以下版本开始：
    1.2

        public List<DataFlavor> getFlavorsForNative(String nat)

        结果
