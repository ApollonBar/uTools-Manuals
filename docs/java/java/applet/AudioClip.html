Interface AudioClip


  • public interface AudioClip
    AudioClip界面是播放声音剪辑的简单抽象。 多个AudioClip项目可以同时播放，并将产生的声音混合在一起以产生复合。
    从以下版本开始：
    JDK1.0

    • 方法摘要

      所有方法  接口方法  抽象方法 
      Modifier and Type Method and Description
      void loop()
      开始循环播放此音频剪辑。
      void play()
      开始播放此音频剪辑。
      void stop()
      停止播放此音频剪辑。
