Package java.applet Description

提供创建applet所需的类以及applet用于与其applet上下文进行通信的类。

小程序框架涉及两个实体： 小程序和小程序上下文 。 小程序是一个可嵌入的窗口（请参阅Panel类），其中包含一些额外的方法，该小程序上下文可用于初始化，启动和停止该小程序。

小程序上下文是负责加载和运行小程序的应用程序。 例如，小程序上下文可以是Web浏览器或小程序开发环境。

从以下版本开始：
JDK1.0
