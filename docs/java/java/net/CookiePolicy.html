Interface CookiePolicy


  • public interface CookiePolicy
    CookiePolicy实现决定哪些Cookie应该被接受，哪些应该被拒绝。 提供了三个预定义的策略实现，即ACCEPT_ALL，ACCEPT_NONE和ACCEPT_ORIGINAL_SERVER。

    参见RFC 2965秒。 3.3和7更详细。

    从以下版本开始：
    1.6
