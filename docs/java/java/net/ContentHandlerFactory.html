Interface ContentHandlerFactory


  • public interface ContentHandlerFactory
    该界面为内容处理程序定义了一个工厂。 此接口的实现应将MIME类型映射到ContentHandler的实例。

    该接口由URLStreamHandler类用于为MIME类型创建一个ContentHandler 。

    从以下版本开始：
    JDK1.0
    另请参见：
