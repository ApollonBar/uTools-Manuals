    public enum SourceVersion

    源代码版本的Java编程语言。 有关特定源版本的信息，请参阅The Java™ Language Specification的相应版本。

    请注意，将添加额外的源版本常量来模拟未来版本的语言。

    从以下版本开始：
    1.6
