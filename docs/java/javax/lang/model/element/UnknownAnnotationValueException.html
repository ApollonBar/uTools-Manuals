    public class UnknownAnnotationValueException
extends UnknownEntityException
    表示遇到未知类型的注释值。 如果语言发展，并且可以在注释中存储新的注释值，则可能会发生这种情况。 可能会抛出一个annotation value visitor来表示访问者是为先前版本的语言创建的。
    从以下版本开始：
    1.6
    另请参见：
