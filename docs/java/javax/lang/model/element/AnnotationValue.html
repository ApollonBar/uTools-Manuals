Interface AnnotationValue


  • public interface AnnotationValue
    表示注释类型元素的值。 值为以下类型之一：
    • 一个原始类型的包装类（如Integer ）
    • String
    • TypeMirror
    • VariableElement （表示枚举常数）
    • AnnotationMirror
    • List<? extends AnnotationValue> （以声明顺序表示元素，如果值为数组）
    从以下版本开始：
    1.6
