Interface WildcardType

  • All Superinterfaces:
    AnnotatedConstruct ， TypeMirror 

    public interface WildcardType
extends TypeMirror
    表示通配符类型参数。 示例包括： 
       ? ? extends Number ? super T

    通配符可以由extends子句显式设置其上限，其下限由super子句显式设置，或者既不（也不是两者）。
