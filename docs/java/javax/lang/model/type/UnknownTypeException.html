    public class UnknownTypeException
extends UnknownEntityException
    表示遇到未知类型的类型。 如果语言演变，并且将新类型添加到TypeMirror层次结构中，则可能会发生这种情况。 可能由type visitor抛出，表示访问者是为先前版本的语言创建的。
    从以下版本开始：
    1.6
    另请参见：
    TypeVisitor.visitUnknown(javax.lang.model.type.TypeMirror, P) ， Serialized Form
