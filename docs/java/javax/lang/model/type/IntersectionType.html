Interface IntersectionType

    public interface IntersectionType
extends TypeMirror
    表示交叉路口类型。

    交集类型可以在程序中隐式或明确声明。 例如，类型参数<T extends Number & Runnable>是（隐式）交集类型。 由于RELEASE_8 ，这是通过一个表示IntersectionType与Number和Runnable作为其边界。

    Implementation Note:
    也是从 RELEASE_8 ，在参考实现中， IntersectionType用于建模转换表达式的显式目标类型。
