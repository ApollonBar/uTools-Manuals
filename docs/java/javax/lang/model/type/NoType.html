    public interface NoType
extends TypeMirror
    在没有实际类型适合的情况下使用的伪类型。 NoType的种类有：
    • VOID - 对应于关键字void 。
    • PACKAGE - 一个包元素的伪类型。
    • NONE - 用于没有实际类型适合的其他情况; 例如，java.lang.Object的java.lang.Object 。
