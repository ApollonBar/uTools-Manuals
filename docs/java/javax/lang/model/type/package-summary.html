Package javax.lang.model.type Description

用于建模Java编程语言类型的接口。

除非在特定实现中另有规定，否则应该预期此程序包中方法返回的集合将被调用者不可修改，并且对并发访问是不安全的。

除非另有说明，在此包中的方法将抛出一个NullPointerException如果给一个null说法。

从以下版本开始：
1.6
