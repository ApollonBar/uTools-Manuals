    工厂创建SchemaFactory 。

    请勿使用本类

    在JSR-206标准化过程中，该课程是早期提案的一部分。 该提案最终被放弃，但这个类意外地留在了源码树中，并进入了最终版本。

    此类不参与任何JAXP 1.3或JAXP 1.4处理。 它不能被用户或JAXP实现使用。
