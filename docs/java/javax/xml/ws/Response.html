    public interface Response<T>
extends Future<T>
    Response接口提供了用于获取响应于操作调用发送的消息的有效载荷和上下文的方法。

    对于异步操作调用，它提供了其他方法来检查请求的状态。 get(...)方法可能会抛出标准异常集，其原因可能是RemoteException或WebServiceException ，表示在异步方法调用期间发生的错误。

    从以下版本开始：
    JAX-WS 2.0
