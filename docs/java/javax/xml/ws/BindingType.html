Annotation Type BindingType


  • @Target(value=TYPE)
 @Retention(value=RUNTIME)

public @interface BindingType
    BindingType注释用于指定用于Web服务端点实现类的绑定。

    取决于正在使用的平台，此注释可能以编程方式或通过部署描述符进行覆盖。

    从以下版本开始：
    JAX-WS 2.0
