Interface Binding

  • All Known Subinterfaces:
    HTTPBinding ， SOAPBinding 

    public interface Binding
    Binding接口是JAX-WS协议绑定的基本接口。
    从以下版本开始：
    JAX-WS 2.0

    • 方法摘要

      所有方法  接口方法  抽象方法

      获取此绑定实例的URI。

    • 方法详细信息

      • getHandlerChain

        List<Handler> getHandlerChain()
        获取协议绑定实例的处理程序链的副本。 如果修改了返回的链，则需要调用setHandlerChain以使用新链来配置绑定实例。
        结果
        java.util.List <Handler>处理程序链

      • setHandlerChain

        void setHandlerChain(List<Handler> chain)
        设置协议绑定实例的处理程序链。
        参数
        chain - 处理程序配置条目列表
        异常
        WebServiceException - 处理程序链的配置错误
        UnsupportedOperationException - 如果不支持此操作。 这样做可以避免任何超前预配置的处理程序链。

      • getBindingID

        String getBindingID()
        获取此绑定实例的URI。
        结果
        String端口的绑定标识符。 永远不会返回null
        从以下版本开始：
        JAX-WS 2.1
