javax.xml.ws

Annotation Type ResponseWrapper


  • @Target(value=METHOD)
 @Retention(value=RUNTIME)
 @Documented
public @interface ResponseWrapper
    用于使用要在运行时使用的响应包装器bean来注释服务端点接口中的方法。 localName的默认值是operationName中定义的WebMethod附注Response和targetNamespace是SEI的目标命名空间。

    当从Java开始时，使用该注解来解决文档文字模式中的重载冲突。 在这种情况下，只需要className 。

    从以下版本开始：
    JAX-WS 2.0
