javax.xml.ws.handler

Interface HandlerResolver


  • public interface HandlerResolver
    HandlerResolver是由应用程序实现的接口，用于在创建代理/调度对象时获取对代理/调度对象上设置的处理程序链的控制。

    A HandlerResolver可以在Service使用setHandlerResolver方法设置。

    当运行时调用HandlerResolver时，它将传递一个包含有关代理/调度对象将要访问的端口的信息的PortInfo对象。

    从以下版本开始：
    JAX-WS 2.0
    另请参见：
    Service.setHandlerResolver(javax.xml.ws.handler.HandlerResolver)
