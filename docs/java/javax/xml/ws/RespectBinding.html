javax.xml.ws

Annotation Type RespectBinding


  • @Target(value={TYPE,METHOD,字段})
 @Retention(value=RUNTIME)
 @Documented
 @WebServiceFeatureAnnotation(id="javax.xml.ws.RespectBindingFeature",
                             bean=RespectBindingFeature.class)
public @interface RespectBinding
    该功能说明了在JAX-WS运行时中使用wsdl:binding 。

    此注释只能在结合使用javax.jws.WebService ， WebServiceProvider ， WebServiceRef个注释。 当与javax.jws.WebService注释一起使用时，此注释必须仅在服务端点实现类上使用。 当与WebServiceRef注释一起使用时，此批注必须仅在创建代理实例时使用。 注入的SEI代理和端点必须遵守RespectBinding注释的值。

    此注释的行为由相应的功能RespectBindingFeature定义。

    从以下版本开始：
    JAX-WS 2.1
