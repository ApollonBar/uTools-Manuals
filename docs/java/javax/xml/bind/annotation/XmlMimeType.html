javax.xml.bind.annotation

Annotation Type XmlMimeType


  • @Retention(value=RUNTIME)
 @Target(value={字段,METHOD,PARAMETER})
public @interface XmlMimeType
    关联控制属性的XML表示形式的MIME类型。

    此注释与数据类型（例如Image或Source）结合使用，这些数据类型与XML中的base64编码二进制文件绑定。

    如果具有此注释的属性具有绑定到xmime：contentType属性的同级属性，并且如果在实例中属性具有值，则该属性的值将优先，并将控制编组。

    从以下版本开始：
    JAXB2.0
