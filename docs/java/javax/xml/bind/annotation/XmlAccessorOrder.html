javax.xml.bind.annotation

Annotation Type XmlAccessorOrder


  • @Inherited
 @Retention(value=RUNTIME)
 @Target(value={PACKAGE,TYPE})
public @interface XmlAccessorOrder

    控制类中字段和属性的顺序。

    用法

    @XmlAccessorOrder注释可以与以下程序元素一起使用：

    • 包
    • 顶级班

    有关其他常见信息，请参阅javax.xml.bind包javadoc中的“包规范”。

    有效的XmlAccessOrder一类确定如下：

    • 如果一个类上有一个@XmlAccessorOrder ，那么它被使用。
    • 否则，如果一个超级类别中存在一个@XmlAccessorOrder ，那么它是继承的（凭借Inherited ）
    • 否则， 该类的包上的@XmlAccessorOrder被使用，如果它在那里。
    • 否则XmlAccessOrder.UNDEFINED 。

    此注释可与以下注释一起使用： XmlType ， XmlRootElement ， XmlAccessorType ， XmlSchema ， XmlSchemaType ， XmlSchemaTypes ， XmlJavaTypeAdapter 。 它也可以在包级别使用以下注释 ： XmlJavaTypeAdapter 。
