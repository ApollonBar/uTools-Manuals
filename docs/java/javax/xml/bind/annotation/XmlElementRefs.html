javax.xml.bind.annotation

Annotation Type XmlElementRefs


  • @Retention(value=RUNTIME)
 @Target(value={字段,METHOD})
public @interface XmlElementRefs
    标记一个涉及XmlElement或JAXBElement的类的属性。

    与元素属性（具有XmlElement注释的属性）相比，引用属性具有不同的替换语义。 当子类分配给属性时，元素属性将生成与@xsi：type相同的标记名，而引用属性会生成不同的标记名称（子类上的标记名称）。

    此注释可与以下注释一起使用： XmlJavaTypeAdapter ， XmlElementWrapper 。

    从以下版本开始：
    JAXB2.0
    另请参见：
    XmlElementWrapper ， XmlElementRef
