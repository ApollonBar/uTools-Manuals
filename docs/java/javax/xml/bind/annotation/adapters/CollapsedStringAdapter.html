  • public class CollapsedStringAdapter
extends XmlAdapter<String,String>
    内置XmlAdapter处理xs:token及其派生类型。

    此适配器删除前导和尾随的空格，然后通过单个空格字符“'截断任何sequnce的选项卡，CR，LF和SP。

    从以下版本开始：
    JAXB 2.0
