javax.xml.bind.annotation

Annotation Type XmlAttachmentRef


  • @Retention(value=RUNTIME)
 @Target(value={字段,METHOD,PARAMETER})
public @interface XmlAttachmentRef
    标记一个字段/属性，它的XML表单是一个uri引用mime内容。 mime内容被最佳地作为附件存储在行外。 字段/属性必须始终映射到DataHandler类。

    用法 

      @XmlRootElement
 class Foo {
   @XmlAttachmentRef
   @XmlAttribute
   DataHandler data;

   @XmlAttachmentRef
   @XmlElement
   DataHandler body;
 }
    以上代码映射到以下XML： 
      <xs:element name="foo" xmlns:ref="http://ws-i.org/profiles/basic/1.1/xsd">
   <xs:complexType>
     <xs:sequence>
       <xs:element name="body" type="ref:swaRef" minOccurs="0" />
     </xs:sequence>
     <xs:attribute name="data" type="ref:swaRef" use="optional" />
   </xs:complexType>
 </xs:element>

    上述绑定支持WS-I AP 1.0 WS-I Attachments Profile Version 1.0.

    从以下版本开始：
    JAXB2.0
