    public abstract static class Unmarshaller.Listener
extends Object

    注册这个类的一个实现的实例与Unmarshaller从外部侦听解组事件。

    此类可以在XML数据未编组的情况下对JAXB映射类的实例进行前处理和后处理。 将XML内容解组为JAXBElement实例或表示复杂类型定义的JAXB映射类时，将调用事件回调。 解组到表示简单类型定义的Java数据类型的实例时，不会调用事件回调。

    外部监听器是定义unmarshal事件回调的两种不同的机制之一。 有关概述，请参阅Unmarshal Event Callbacks 。

      • beforeUnmarshal

        public void beforeUnmarshal(Object target,
                            Object parent)

        在解组之前调用回调方法到target 。

        此方法在创建target之后立即被调用，并且此对象的解组开始之前。 请注意，如果target的类定义了自己的beforeUnmarshal方法，则在调用此方法之前调用类特定的回调方法。
        参数
        target - 在解组之前JAXB映射类的非空实例。
        parent - JAXB映射类的实例，最终将引用target 。 null当target是根元素。

      • afterUnmarshal

        public void afterUnmarshal(Object target,
                           Object parent)

        将解析XML数据解组后调用回调方法为target 。

        所有这些属性（IDREF除外）都被归并到target中 ，但在target被设置为parent对象之前调用此方法。 请注意，如果target的类定义了自己的afterUnmarshal方法，则在调用此方法之前调用类特定的回调方法。
        参数
        target - 在解组之前JAXB映射类的非空实例。
        parent - JAXB映射类的实例，将引用target 。 null当target是根元素。
