    public abstract class AbstractMarshallerImpl
extends Object
implements Marshaller
    部分默认Marshaller实现。

    这个类提供了部分默认实现Marshaller接口。

    一个JAXB提供者必须实现的唯一方法是marshal(Object, javax.xml.transform.Result) ， marshal(Object, javax.xml.stream.XMLStreamWriter)和marshal(Object, javax.xml.stream.XMLEventWriter) 。

    从以下版本开始：
    JAXB1.0
