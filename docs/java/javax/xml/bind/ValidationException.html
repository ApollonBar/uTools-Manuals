    public class ValidationException
extends JAXBException
    此异常表示执行验证操作时发生错误。

    ValidationEventHandler可能导致在验证操作期间抛出此异常。 见ValidationEventHandler.handleEvent(ValidationEvent) 。

    从以下版本开始：
    JAXB1.0
    另请参见：
