      • report

        void report(String message,
            String errorType,
            Object relatedInformation,
            Location location)
     throws XMLStreamException
        以应用程序特定格式报告所需的消息。 只能通过此界面报告警告和非致命错误。 致命错误应抛出XMLStreamException。
        参数
        message - 错误信息
        errorType - 实现定义的错误类型
        relatedInformation - 与错误相关的信息（如果有）
        location - 错误的位置（如果有）
        异常
        XMLStreamException
