Package javax.xml.stream

  • 接口摘要  
    接口 描述
    EventFilter
    该接口声明一个可以创建过滤XMLEventReaders的简单过滤器接口
    Location
    提供有关活动位置的信息。
    StreamFilter
    此接口声明可以创建过滤XMLStreamReaders的简单过滤器接口
    XMLEventReader
    这是解析XML事件的顶级界面。
    XMLEventWriter
    这是编写XML文档的顶级界面。
    XMLReporter
    此界面用于报告非致命错误。
    XMLResolver
    此接口用于在XML解析期间解析资源。
    XMLStreamConstants
    此接口声明此API中使用的常量。
    XMLStreamReader
    XMLStreamReader接口允许转发，只读访问XML。
    XMLStreamWriter
    XMLStreamWriter接口指定如何编写XML。
  • 类摘要  
    类 描述
    XMLEventFactory
    该界面定义了一个用于创建XMLEvents实例的实用程序类
    XMLInputFactory
    定义了一个工厂的抽象实现来获取流。
    XMLOutputFactory
    定义用于获取XMLEventWriters和XMLStreamWrit的工厂的抽象实现。
