      • accept

        boolean accept(XMLStreamReader reader)
        测试当前状态是否是此流的一部分。 如果此过滤器接受此事件，则此方法将返回true，否则返回false。 该方法在接受状态时不应改变读者的状态。
        参数
        reader - 要测试的事件
        结果
        true if this filter accepts this event, false otherwise
