Interface Location


  • public interface Location
    提供有关活动位置的信息。 位置提供的所有信息都是可选的。 例如，应用程序只能报告行号。
    从以下版本开始：
    1.6
    Version:
    1.0

      • getCharacterOffset

        int getCharacterOffset()
        将字节或字符偏移量返回到此位置指向的输入源。 如果输入源是文件或字节流，那么这是该流中的字节偏移，但是如果输入源是字符介质，则偏移量是字符偏移量。 如果没有可用的偏移量，则返回-1。
        结果
        当前的偏移量
