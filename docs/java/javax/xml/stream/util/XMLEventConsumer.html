    XMLEventWriter 

    public interface XMLEventConsumer
    该接口定义事件消费者接口。 消费者的合同是接受事件。 该接口可用于将对象标记为能够接收事件。 可以立即连续调用多次添加，因此消费者必须能够缓存尚未处理的事件。

      • add

        void add(XMLEvent event)
  throws XMLStreamException
        此方法会向消费者添加一个事件。 调用此方法会使事件参数无效。 客户端应用程序应该在调用add时放弃对此事件的所有引用。 继续使用这些引用的应用程序的行为是未定义的。
        参数
        event - 要添加的事件，可能不为null
        异常
        XMLStreamException
