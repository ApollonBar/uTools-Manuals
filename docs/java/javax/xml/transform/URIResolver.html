    • 方法详细信息

      • resolve

        Source resolve(String href,
               String base)
        throws TransformerException
        处理器遇到xsl：include，xsl：import或document（）函数时调用。
        参数
        href - 一个href属性，可以是相对的或绝对的。
        base - 如果绝对URI是必需的，第一个参数将被绝对的基本URI。
        结果
        Source对象，如果href无法解析，则为null，并且处理器应尝试解析URI本身。
        异常
        TransformerException - if an error occurs when trying to resolve the URI.
