Interface TemplatesHandler

  • All Superinterfaces:
    ContentHandler 

    public interface TemplatesHandler
extends ContentHandler
    可用于将SAX解析事件（解析变换指令）转换为“模板”对象的SAX ContentHandler。

    请注意，TemplatesHandler不需要实现LexicalHandler。

    • 方法详细信息

      • getTemplates

        Templates getTemplates()
        当将TemplatesHandler对象用作解析转换指令的ContentHandler时，它将创建一个Templates对象，一旦SAX事件完成，调用者就可以获取该对象。
        结果
        在SAX事件过程中创建的Templates对象，如果没有创建Templates对象，则为null。

      • setSystemId

        void setSystemId(String systemID)
        设置由此构建器创建的Templates对象的基本ID（URI或系统ID）。 必须设置此值才能解析样式表中的相对URI。 这必须在startDocument事件之前调用。
        参数
        systemID - 此样式表的基本URI。
