    • 字段详细信息

      • PI_DISABLE_OUTPUT_ESCAPING

        static final String PI_DISABLE_OUTPUT_ESCAPING
        如果结果树禁用输出转义，则发送的处理指令的名称。

        通常，结果树序列化在输出文本节点时会转义＆和<（和其他字符）。 这样可以确保输出格式良好。 然而，有时候可以方便地生成几乎不是格式好的XML的输出; 例如，输出可以包括将被随后的非XML感知过程转换成良好格式的XML的不合格部分。 如果使用此名称发送处理指令，则应该输出序列化，而不进行任何转义。

        结果DOM树也可以将PI_DISABLE_OUTPUT_ESCAPING和PI_ENABLE_OUTPUT_ESCAPING插入到树中。

        另请参见：
        disable-output-escaping in XSLT Specification ， Constant Field Values

      • PI_ENABLE_OUTPUT_ESCAPING

    • 方法详细信息

      • setSystemId

        void setSystemId(String systemId)
        设置此结果的系统标识符。

        如果结果不被写入文件，系统标识符是可选的。 但是，应用程序可能仍然希望提供一个用于错误消息和警告，或者解决相对输出标识符。

        参数
        systemId - 系统标识符作为URI字符串。

      • getSystemId

        结果
        使用setSystemId设置的系统标识符，如果未调用setSystemId，则为null。
