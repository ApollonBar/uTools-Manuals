    • 方法详细信息

      • setSystemId

        void setSystemId(String systemId)
        设置此源的系统标识符。

        如果源不从URL获取其数据，则系统标识符是可选的，但是提供一个可能仍然有用。 应用程序可以使用系统标识符，例如，解析相对URI并包含在错误消息和警告中。

        参数
        systemId - 系统标识符作为URL字符串。

      • getSystemId

        String getSystemId()
        获取使用setSystemId设置的系统标识符。
        结果
        使用setSystemId设置的系统标识符，如果未调用setSystemId，则为null。
