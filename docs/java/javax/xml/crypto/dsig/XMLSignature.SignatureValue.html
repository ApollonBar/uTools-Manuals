    public static interface XMLSignature.SignatureValue
extends XMLStructure
    W3C Recommendation for XML-Signature Syntax and Processing中定义的XML SignatureValue元素的表示 。 XML模式定义定义为： 

      <element name="SignatureValue" type="ds:SignatureValueType"/>
     <complexType name="SignatureValueType">
       <simpleContent>
         <extension base="base64Binary">
           <attribute name="Id" type="ID" use="optional"/>
         </extension>
       </simpleContent>
     </complexType>

      • validate

        boolean validate(XMLValidateContext validateContext)
          throws XMLSignatureException
        验证签名值。 此方法执行在所计算出的签名的加密验证SignedInfo所述的XMLSignature 。

        该方法仅在第一次调用时验证签名。 在随后的调用中，它返回缓存的结果。

        参数
        validateContext - 验证上下文
        结果
        true如果签名验证成功; false否则
        异常
        NullPointerException - 如果 validateContext是 null
        XMLSignatureException - 如果在验证签名时发生意外异常
