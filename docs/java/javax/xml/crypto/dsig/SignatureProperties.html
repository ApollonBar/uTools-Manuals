Interface SignatureProperties

  • All Superinterfaces:
    XMLStructure 

    public interface SignatureProperties
extends XMLStructure
    在W3C Recommendation for XML-Signature Syntax and Processing中定义的XML SignatureProperties元素的表示 。 XML模式定义定义为： 
       <element name="SignatureProperties" type="ds:SignaturePropertiesType"/> <complexType name="SignaturePropertiesType"> <sequence> <element ref="ds:SignatureProperty" maxOccurs="unbounded"/> </sequence> <attribute name="Id" type="ID" use="optional"/> </complexType>
    可以通过调用XMLSignatureFactory类的newSignatureProperties方法创建一个SignatureProperties实例; 例如： 
      XMLSignatureFactory factory = XMLSignatureFactory.getInstance("DOM");
   SignatureProperties properties =
      factory.newSignatureProperties(props, "signature-properties-1");
    从以下版本开始：
    1.6
    另请参见：
    XMLSignatureFactory.newSignatureProperties(List, String) ， SignatureProperty
