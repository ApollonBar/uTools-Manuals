Interface DigestMethod

  • All Superinterfaces:
    AlgorithmMethod ， XMLStructure 

    public interface DigestMethod
extends XMLStructure, AlgorithmMethod
    W3C Recommendation for XML-Signature Syntax and Processing中定义的XML DigestMethod元素的表示 。 XML模式定义定义为： 

      <element name="DigestMethod" type="ds:DigestMethodType"/>
     <complexType name="DigestMethodType" mixed="true">
       <sequence>
         <any namespace="##any" minOccurs="0" maxOccurs="unbounded"/>
           <!-- (0,unbounded) elements from (1,1) namespace -->
       </sequence>
       <attribute name="Algorithm" type="anyURI" use="required"/>
     </complexType>
    可以通过调用XMLSignatureFactory类的newDigestMethod方法来创建一个DigestMethod实例。
    从以下版本开始：
    1.6
    另请参见：
    XMLSignatureFactory.newDigestMethod(String, DigestMethodParameterSpec)
