Interface URIDereferencer


  • public interface URIDereferencer
    URIReference的取消引用 。

    取消引用URIReference的URIReference是OctetStreamData或NodeSetData的一个实例。 除非URIReference是与 XML签名语法和处理W3C建议书第4.2节中定义的同一文献参考 ，否则URIReference的URIReference结果必须为OctetStreamData 。

    从以下版本开始：
    1.6
    另请参见：
    XMLCryptoContext.setURIDereferencer(URIDereferencer) ， XMLCryptoContext.getURIDereferencer()
