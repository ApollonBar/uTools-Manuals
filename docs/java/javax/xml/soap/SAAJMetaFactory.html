  • public abstract class SAAJMetaFactory
extends Object
    在SAAJ API中定义的工厂的实现类的接入点。 所有在SAAJ 1.3 newInstance定义的newInstance方法都适用于该类的实例，以便实际创建对象。 SAAJ 1.2中存在的newInstance()方法（在SOAPFactory和MessageFactory中）的实现已经被更新，当SAAJ 1.2定义的查找找不到Factory实现类名时，它们也被委派给SAAJMetaFactory。

    SAAJMetaFactory是一个服务提供者界面。 这个班没有公开的方法。

    从以下版本开始：
    SAAJ 1.3
