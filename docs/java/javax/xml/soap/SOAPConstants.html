      • DYNAMIC_SOAP_PROTOCOL

        static final String DYNAMIC_SOAP_PROTOCOL
        用于创建MessageFactory实例，创建SOAPMessages ，其具体类型基于传递给createMessage方法的Content-Type MIME标头。 如果没有Content-Type头被传递，则createMessage可能抛出IllegalArgumentException ，或者在的不带参数的版本的情况下createMessage ，一个UnsupportedOperationException 。
        从以下版本开始：
        SAAJ 1.3

      • URI_NS_SOAP_ENCODING

        static final String URI_NS_SOAP_ENCODING
        SOAP 1.1编码的命名空间标识符。 URI_NS_SOAP_ENVELOPE命名encodingStyle中的一个名称为encodingStyle的URI_NS_SOAP_ENVELOPE ，并设置为值URI_NS_SOAP_ENCODING ，可以添加到元素中以指示它使用SOAP 1.1规范第5节中的规则进行编码。

      • URI_SOAP_ACTOR_NEXT

        static final String URI_SOAP_ACTOR_NEXT
        URI标识下一个处理SOAP请求的应用程序作为SOAP 1.1标头条目的预期actor（参见SOAP 1.1规范的第4.2.2节）。

        这个值可以被传递到SOAPHeader.examineMustUnderstandHeaderElements(String) ， SOAPHeader.examineHeaderElements(String)和SOAPHeader.extractHeaderElements(String)
