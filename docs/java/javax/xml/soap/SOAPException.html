    public class SOAPException
extends 异常
    发生SOAP异常发生的异常。 SOAPException对象可能包含一个String ，它给出了String的原因，一个嵌入的Throwable对象，或者两者。 该类提供检索原因消息和检索嵌入式Throwable对象的方法。

    抛出SOAPException对象的典型原因是诸如难以设置标题，无法发送消息，无法与提供商建立连接的问题。 嵌入Throwable对象的原因包括诸如输入/输出错误或解析问题之类的问题，例如解析头部的错误。
