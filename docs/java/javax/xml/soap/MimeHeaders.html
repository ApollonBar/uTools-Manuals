  • public class MimeHeaders
extends Object
    一个用于MimeHeader对象的MimeHeader ，它们表示消息的MIME部分中存在的MIME头。

    此类主要用于应用程序根据某些MIME标题和值检索特定附件。 这个类最有可能被执行的AttachmentPart和SAAJ API的其他依赖于MIME的部分使用。

    另请参见：
    SOAPMessage.getAttachments() ， AttachmentPart

      • setHeader

        public void setHeader(String name,
                      String value)
        将名称与给定名称匹配的第一个标题条目的当前值替换为给定值，如果没有现有的标题名称匹配，则添加一个新标题。 此方法也会在第一个之后删除所有匹配的头文件。

        请注意，RFC822头文件只能包含US-ASCII字符。

        参数
        name - 一个 String ，具有要搜索的标题的名称
        value - 一个 String ，其值将替换指定头的当前值
        异常
        IllegalArgumentException - 如果在mime标题名称或正在设置的值中出现问题
        另请参见：
        getHeader(java.lang.String)
