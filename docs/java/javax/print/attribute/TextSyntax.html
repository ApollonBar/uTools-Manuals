    public abstract class TextSyntax

    Class TextSyntax是一个抽象基类，提供了值为字符串的所有属性的常见实现。 文本属性包括一个指示自然语言的区域设置。 因此，文本属性始终表示本地化的字符串。 一旦构造，文本属性的值是不可变的。
