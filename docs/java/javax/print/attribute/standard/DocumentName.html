    DocumentName类是一个打印属性类，一个指定文档名称的文本属性。 DocumentName是打印数据（文档）的属性，而不是打印作业。 文档的名称是由客户端定义的任意字符串。 但是，如果未指定JobName，则应使用DocumentName，这意味着DocumentName的支持规范需要报告JobName，反之亦然。 见JobName以获取更多信息。

    IPP兼容性：字符串值给出IPP名称值。 该地区提供IPP自然语言。 getName()返回的类别名称提供IPP属性名称。
