    public class JobState
extends EnumSyntax
implements PrintJobAttribute
    JobState是一个打印属性类，一个枚举，用于标识打印作业的当前状态。 JobState类定义了标准作业状态值。 打印服务实施只需要报告适合特定实施的工作状态; 它不必报告每个定义的工作状态。 JobStateReasons属性增加了JobState属性，以给出在给定作业状态下有关作业的更详细信息。

    IPP兼容性：由返回的类别名称getName()是IPP属性名称。 枚举的整数值是IPP枚举值。 toString()方法返回属性值的IPP字符串表示形式。

      • PROCESSING

        public static final JobState PROCESSING
        工作正在处理中。 正在发生以下一个或多个活动：
        1. 该作业正在或正试图使用一个或多个纯软件进程，分析，创建或解释PDL等。

        2. 该作业正在使用或正在尝试使用一个或多个解释PDL的硬件设备，在介质上进行标记和/或执行整理，例如装订等。

        3. 打印机已经准备好打印作业，但输出设备尚未打印，因为作业尚未到达输出设备，或者因为作业在输出设备或其他后台处理程序中排队等待输出设备打印它。

        当作业处于PROCESSING状态时，整个作业状态包括在打印机的PrinterState和PrinterStateReasons属性中表示的详细状态。

        尽管实施方式不需要，但在作业的JobStateReasons属性中可以包含附加值，以指示作业的进度，例如添加JOB_PRINTING值以指示输出设备何时实际在纸张上标记和/或PROCESSING_TO_STOP_POINT值来指示打印机正在取消或中止工作。

      • PROCESSING_STOPPED

        public static final JobState PROCESSING_STOPPED
        该工作由于任何原因处理而停止，并将在原因不再存在的情况下返回到PROCESSING状态。

        作业的JobStateReasons属性可能指示作业停止处理的原因。 例如，如果输出设备停止，则PRINTER_STOPPED值可能包含在作业的JobStateReasons属性中。

        注意：当输出设备停止运行时，设备通常在设备本地以人类可读的形式指出其状态。 通过查询打印机的PrinterState和PrinterStateReasons属性，客户端可以远程获取更完整的设备状态。

      • CANCELED

        public static final JobState CANCELED
        这个工作已被一些人力机构取消，打印机已经完成了取消工作，所有工作状态属性都达到了最终的工作价值。 当打印机取消作业时，作业保持当前状态，但作业的JobStateReasons属性应包含PROCESSING_TO_STOP_POINT值和CANCELED_BY_USER，CANCELED_BY_OPERATOR或CANCELED_AT_DEVICE值之一。 当作业移动到CANCELED状态时，必须删除PROCESSING_TO_STOP_POINT值（如果存在），但必须保留CANCELED_BY_ xxx值（如果存在）。

      • ABORTED

        public static final JobState ABORTED
        该作业已被系统中止（通常在作业处于PROCESSING或PROCESSING_STOPPED状态时），打印机已完成中止作业，并且所有作业状态属性已达到作业的最终值。 当打印机正在中止作业时，作业将保持当前状态，但作业的JobStateReasons属性应包含PROCESSING_TO_STOP_POINT和ABORTED_BY_SYSTEM值。 当作业移动到ABORTED状态时，必须删除PROCESSING_TO_STOP_POINT值（如果存在），但必须保留ABORTED_BY_SYSTEM值（如果存在）。

      • COMPLETED

        public static final JobState COMPLETED
        作业已成功完成或处理完毕后出现警告或错误，所有作业介质单已经成功堆放在相应的输出仓中，并且所有作业状态属性已达到作业的最终值。 作业的JobStateReasons属性应包含以下值之一：COMPLETED_SUCCESSFULLY，COMPLETED_WITH_WARNINGS或COMPLETED_WITH_ERRORS。
