    public class MediaTray
extends Media
implements Attribute
    MediaTray类是Media的一个子类。 MediaTray类是一个打印属性类，一个枚举，用于指定作业的介质托盘或bin。 可以使用此属性，而不是指定MediaSize或MediaName。

    MediaTray类为标准媒体类型值声明关键字。 媒体类型属性的实现或站点定义的名称也可以通过定义MediaTray类的子类来创建。

    IPP兼容性： MediaTray是名为纸盘的IPP“media”属性值的表示类。
