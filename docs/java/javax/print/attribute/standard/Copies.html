    Class Copies是一个整数值的打印属性类，它指定要打印的份数。

    在许多设备上，支持的分页副本数量将受到设备上物理输出信箱数量的限制，并且可能与可支持的未分类副本的数量有所不同。

    一个副本属性随n在多文档打印作业（具有多个文档的作业）的值的影响取决于的（可能是默认）值MultipleDocumentHandling属性：

    • SINGLE_DOCUMENT - 结果将是包含所有输入文档的单个输出文档的n个副本。

    • SINGLE_DOCUMENT_NEW_SHEET - 结果将是包含所有输入文档的单个输出文档的n个副本，并且每个输入文档的第一印象将始终从新的介质页开始。

    • SEPARATE_DOCUMENTS_UNCOLLATED_COPIES -结果将是n的第一输入文件的拷贝，然后由第二输入文件的n个拷贝，。 。 。 其次是最后一个输入文档的n个副本。

    • SEPARATE_DOCUMENTS_COLLATED_COPIES - 结果将是第一个输入文档，第二个输入文档。 。 。 最后一个输入文件，该组文件重复n次。

    IPP兼容性：整数值给出IPP整数值。 getName()返回的类别名称提供IPP属性名称。
