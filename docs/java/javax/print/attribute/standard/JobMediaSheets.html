    JobMediaSheets类是一个整数值打印属性类，它指定要为此作业生成的介质页的总数。

    JobMediaSheets属性描述作业的大小。 这个属性不是一个计数器; 如果已知的话，它是有用的路由和调度信息。 如果打印请求中未提供JobMediaSheets属性的值，打印机可能会尝试计算。 即使客户端确实为打印请求中的JobMediaSheets属性提供了一个值，如果打印机能够计算比客户端提供的值更准确的值，打印机也可以选择更改该值。 打印机可能能够在作业提交时间或任何后来的时间点确定JobMediaSheets属性的正确值。

    不同于JobKOctets个JobImpressions属性，则JobMediaSheets值必须包含贡献的由指定的份数的乘法因子Copies属性和指令嵌入到文档数据“副本数量”，如果有的话。 这种差异允许系统管理员使用JobKOctetsSupported和JobImpressionsSupported控制文档的大小（1）和（2）具有JobMediaSheetsSupported的作业大小的下限和上限 。

    IPP兼容性：整数值给出IPP整数值。 getName()返回的类别名称提供IPP属性名称。
