    类NumberUpSupported是打印属性类，一个整数集，给出了支持值一个NumberUp属性。

    IPP兼容性： NumberUpSupported属性的规范数组形式为IPP“number-up-supported”属性中包含的每个数字范围提供了下限和上限。 有关规范数组形式的说明，请参见SetOfIntegerSyntax类。 由getName()返回的类别名称提供IPP属性名称。
