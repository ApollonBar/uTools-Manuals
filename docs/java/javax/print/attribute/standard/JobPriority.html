    JobPriority类是一个整数值打印属性类，用于指定打印作业的优先级。

    如果为打印作业指定了JobPriority属性，则它指定了调度作业的优先级。 较高的值指定较高的优先级。 值1表示最低可能的优先级。 值100表示最高可能的优先级。 在准备打印的作业中，打印机必须打印优先级为n的所有作业，然后打印所有n的优先级值为n -1的作业。

    如果客户端没有为打印作业指定JobPriority属性，并且打印机确实支持JobPriority属性，则打印机必须使用实现定义的默认JobPriority值。

    客户端可以随时为作业指定任何作业优先级值从1到100。 但是，打印服务实例可能支持少于100个不同的作业优先级。 如果是这种情况，打印服务实例会自动将客户机指定的作业优先级值映射到所支持的作业优先级之一，将可用作业优先级之间的100个作业优先级值平均分配。

    IPP兼容性：整数值给出IPP整数值。 由getName()返回的类别名称提供IPP属性名称。
