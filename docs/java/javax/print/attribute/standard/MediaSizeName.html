    public class MediaSizeName
extends Media
    MediaSizeName类是Media的子类。

    可以使用此属性，而不是指定MediaName或MediaTray。

    MediaSizeName类目前声明了几个标准的媒体名称值。

    IPP兼容性： MediaSizeName是用于指定媒体大小的IPP“media”属性的值的表示类。 媒体大小的名称对应于IPP 1.1 RFC RFC 2911中的名称
