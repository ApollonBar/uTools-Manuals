    public final class QueuedJobCount

    Class QueuedJobCount是一个整数值打印属性，指示打印机中的作业数量，其中JobState为PENDING，PENDING_HELD，PROCESSING或PROCESSING_STOPPED。

    IPP兼容性：整数值给出IPP整数值。 由getName()返回的类别名称提供IPP属性名称。
