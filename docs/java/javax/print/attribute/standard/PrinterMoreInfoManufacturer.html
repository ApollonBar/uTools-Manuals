    public final class PrinterMoreInfoManufacturer
extends URISyntax
implements PrintServiceAttribute
    PrinterMoreInfoManufacturer类是打印属性类（URI），用于获取有关此类型设备的更多信息。 从该URI获取的信息用于最终用户消费。 可以从该URI访问Print Service API范围之外的功能（例如，最新的固件，升级，服务代理，可用的可用功能，颜色支持的详细信息）。 该信息旨在与这种打印机密切相关，而不考虑站点特定的修改或服务。

    相比之下， PrinterMoreInfo属性用于查找有关此特定打印机的更多信息，而不是这种一般打印机。

    IPP兼容性：由toString()返回的字符串形式给出了IPP uri值。 由getName()返回的类别名称提供IPP属性名称。
