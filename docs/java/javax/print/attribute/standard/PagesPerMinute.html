    public final class PagesPerMinute
extends IntegerSyntax
implements PrintServiceAttribute
    Class PagesPerMinute是一个整数值打印属性，它表示可能由此打印机生成的最接近的整数的每分钟的标称页数（例如单面，黑白）。 此属性信息丰富，不是服务保证。 一般来说，这是营销文献中用来描述设备的价值。 值为0表示处理页面需要两分多分钟的设备。

    IPP兼容性：整数值给出IPP整数值。 由getName()返回的类别名称提供IPP属性名称。
