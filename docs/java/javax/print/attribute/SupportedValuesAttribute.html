    public interface SupportedValuesAttribute
extends Attribute
    Interface SupportedValuesAttribute是打印属性类实现的标记接口，用于指示属性描述了另一个属性的支持值。 例如，如果打印服务实例支持Copies属性，打印服务实例将具有CopiesSupported属性，该属性是一个SupportedValuesAttribute，给出客户端可以为Copies属性指定的合法值。
