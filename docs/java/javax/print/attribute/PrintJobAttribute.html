    public interface PrintJobAttribute
extends Attribute
    PrintJobAttribute是打印属性类实现的标记界面，用于指示属性描述打印作业的状态或打印作业的其他一些特性。 打印服务实例向打印作业的属性集添加了一些PrintJobAttributes以报告打印作业的状态。 如果一个属性实现了PrintRequestAttribute以及PrintJobAttribute，则客户端可以在属性集中包含属性，以指定属性的值作为打印作业。

    另请参见：
    PrintRequestAttributeSet ， PrintJobAttributeSet
