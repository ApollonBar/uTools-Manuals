    接口DocAttribute是打印属性类实现的标记接口，用于指示属性表示文档的设置。 （“文件”是一个短，易于发音的术语，意思是“一部分打印数据”。）的客户端可以包括在的DocAttribute一个Doc的属性设置为指定的文档的特性。 如果一个属性实现了PrintRequestAttribute以及DocAttribute，那么客户端可能会在属性集中包含属性，该属性指定打印作业，以指定该作业中所有文档的特征。
