javax.print

Interface URIException


  • public interface URIException
    接口URIException是一个mixin接口， PrintException可以实现一个子类来报告涉及URI地址的错误条件。 Print Service API没有定义实现接口URIException的任何打印异常类，这些类别由Print Service实现者自行决定。
