    public abstract class PrintJobAdapter

    用于接收打印作业事件的抽象适配器类。 这个类中的方法是空的。 此类存在为创建侦听器对象的方便。 扩展此类以创建一个PrintJobEvent监听器并覆盖感兴趣的事件的方法。 与ComponentListener接口不同，此抽象接口提供了空方法，因此您只需要定义所需的方法，而不是所有方法。
