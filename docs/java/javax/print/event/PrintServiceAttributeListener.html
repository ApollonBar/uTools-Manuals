Interface PrintServiceAttributeListener


  • public interface PrintServiceAttributeListener
    此监听器接口的实现被附加到PrintService以监视打印服务的状态。

    要监视特定的工作，请参阅PrintJobListener和PrintJobAttributeListener 。

      • attributeUpdate

        void attributeUpdate(PrintServiceAttributeEvent psae)
        被要求在打印服务中通知侦听器事件。 该服务将在事件通知线程上调用此方法。 客户端不应该在此回调中执行冗长的处理，否则可能会阻止事件通知。
        参数
        psae - 事件被通知
