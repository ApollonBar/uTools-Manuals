Interface RowSetListener

  • All Superinterfaces:
    EventListener 

    public interface RowSetListener
extends EventListener
    一个接口，必须由一个组件实现，该组件希望在RowSet对象的生命中发生重大事件时被通知。 通过RowSet.addRowSetListener方法RowSet.addRowSetListener RowSet对象注册，组件成为侦听器。 注册组件如何实现此接口确定在通知事件时的功能。
    从以下版本开始：
    1.4
