    public class ConnectionEvent
extends EventObject

    一个Event对象，提供有关连接相关事件源的信息。 ConnectionEvent对象在应用程序关闭池化连接和发生错误时生成。 ConnectionEvent对象包含两种信息：

    • 合并的连接由应用程序关闭
    • 在发生错误的情况下， SQLException即将被抛出到应用程序中
