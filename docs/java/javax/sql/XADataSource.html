Interface XADataSource

  • All Superinterfaces:
    CommonDataSource 

    public interface XADataSource
extends CommonDataSource
    一个用于XAConnection对象的工厂。 实现XADataSource接口的对象通常使用使用Java命名和目录接口（JNDI）的命名服务注册。

    XADataSource的XADataSource必须包括一个public no-arg构造函数。

    从以下版本开始：
    1.4

      • getXAConnection

        XAConnection getXAConnection()
                      throws SQLException
        尝试建立可在分布式事务中使用的物理数据库连接。
        结果
        一个 XAConnection与数据源的物理连接的 XAConnection对象，可以在分布式事务中使用
        异常
        SQLException - 如果发生数据库访问错误
        SQLFeatureNotSupportedException - 如果JDBC驱动程序不支持此方法
        SQLTimeoutException - 当驱动程序确定已经超过了 setLoginTimeout方法指定的超时值，并且至少尝试取消当前的数据库连接尝试
        从以下版本开始：
        1.4

      • getXAConnection

        XAConnection getXAConnection(String user,
                             String password)
                      throws SQLException
        尝试使用给定的用户名和密码建立物理数据库连接。 返回的连接是可以在分布式事务中使用的连接。
        参数
        user - 正在连接的数据库用户
        password - 用户密码
        结果
        一个 XAConnection对象，表示与数据源的物理连接，可以在分布式事务中使用
        异常
        SQLException - 如果发生数据库访问错误
        SQLFeatureNotSupportedException - 如果JDBC驱动程序不支持此方法
        SQLTimeoutException - 当驱动程序确定已经超过 setLoginTimeout方法指定的超时值，并且至少尝试取消当前数据库连接尝试
        从以下版本开始：
        1.4
