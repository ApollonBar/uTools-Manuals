    public class SerialRef

    一个Ref对象的序列化映射，它是Java编程语言中的一个SQL REF值的映射。

    SerialRef类提供了一个从Ref对象创建SerialRef实例的Ref函数，并提供了获取和设置Ref对象的方法。

    线程安全

    SerialRef对于多个并发线程使用是不安全的。 如果SerialRef由多个线程使用，则应通过适当的同步来控制对SerialRef的访问。
