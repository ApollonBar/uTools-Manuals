Interface XAConnection

  • All Superinterfaces:
    PooledConnection 

    public interface XAConnection
extends PooledConnection
    为分布式事务提供支持的对象。 一个XAConnection对象可以在分布式事务由一个来征XAResource对象。 事务管理器，通常是中间层服务器的一部分，通过XAConnection对象来管理一个XAResource对象。

    应用程序员不直接使用此接口; 相反，它由在中间层服务器中工作的事务管理器使用。

    从以下版本开始：
    1.4
