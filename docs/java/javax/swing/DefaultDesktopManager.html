    Serializable ， DesktopManager 

    public class DefaultDesktopManager
extends Object
implements DesktopManager, Serializable
    这是DesktopManager的一个DesktopManager 。 目前，它实现了基本行为管理JInternalFrame在任意父秒。 JInternalFrame的孩子的JDesktop将使用这个组件来处理他们的类似桌面的动作。

    这个类为各种JInternalFrame方法提供了一个策略，它不是直接调用，而是将各种JInternalFrame方法调用到DesktopManager中。

    另请参见：
    JDesktopPane ， JInternalFrame
