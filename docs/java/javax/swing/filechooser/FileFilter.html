    public abstract class FileFilter

    FileFilter是JFileChooser用于过滤向用户显示的文件集的抽象类。 见FileNameExtensionFilter该滤波器使用文件扩展名的实现。

    一个FileFilter可以在设置JFileChooser从出现在目录列表保存不需要的文件。 有关简单文件过滤器的示例实现，请参阅yourJDK/demo/jfc/FileChooserDemo/ExampleFileFilter.java 。 有关更多信息和示例，请参阅How to Use File Choosers “Java教程 ”中的一节。

    另请参见：
    FileNameExtensionFilter ， JFileChooser.setFileFilter(javax.swing.filechooser.FileFilter) ， JFileChooser.addChoosableFileFilter(javax.swing.filechooser.FileFilter)
