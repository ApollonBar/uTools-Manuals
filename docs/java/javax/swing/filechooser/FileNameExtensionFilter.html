Class FileNameExtensionFilter

  • java.lang.Object
    • javax.swing.filechooser.FileFilter
      • javax.swing.filechooser.FileNameExtensionFilter

  • public final class FileNameExtensionFilter
extends FileFilter
    FileFilter的一个实现，使用指定的一组FileFilter进行过滤。 文件的扩展名是最后一个“。”之后的文件名的一部分。 名称不包含“。”的文件 没有文件扩展名。 文件扩展名比较不区分大小写。

    以下示例创建一个FileNameExtensionFilter将显示jpg文件： 

      FileFilter filter = new FileNameExtensionFilter("JPEG file", "jpg", "jpeg");
 JFileChooser fileChooser = ...;
 fileChooser.addChoosableFileFilter(filter);
    从以下版本开始：
    1.6
