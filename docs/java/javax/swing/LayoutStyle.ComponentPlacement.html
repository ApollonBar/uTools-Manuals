      • UNRELATED

        指示两个组件的枚举值在视觉上不相关，并将放置在同一个父项中。 例如，组件的分组通常是视觉分离的; 常数UNRELATED用于此。

      • INDENT

        public static final LayoutStyle.ComponentPlacement INDENT
        正在请求指示缩进组件的距离的枚举值。 例如，通常，标签的子项将从标签中水平缩进。 要确定这种间隙的首选距离，请使用INDENT型。

        该值通常仅适用于EAST或WEST 。
