    protected class JTree.AccessibleJTree
extends JComponent.AccessibleJComponent
implements AccessibleSelection, TreeSelectionListener, TreeModelListener, TreeExpansionListener
    这个类实现了可访问性支持JTree类。 它提供了适用于树形用户界面元素的Java Accessibility API的实现。

    警告：此类的序列化对象与将来的Swing版本不兼容。 当前的序列化支持适用于运行相同版本的Swing的应用程序之间的短期存储或RMI。 从1.4开始，支持所有JavaBeans的长期存储已经添加到java.beans包中。 请参阅XMLEncoder 。
