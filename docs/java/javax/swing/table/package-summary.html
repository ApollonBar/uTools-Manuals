Package javax.swing.table Description

提供处理javax.swing.JTable类和javax.swing.JTable 。 JTable是Swing的网格或表格视图，用于构建应用程序内的表格数据结构的用户界面。 如果您想要控制表的构造，更新和呈现以及如何查看和管理与表关联的数据，请使用此软件包。

注意：大多数Swing API 不是线程安全的。 有关详细信息，请参阅Threads and Swing ，在部分The Java Tutorial 。
