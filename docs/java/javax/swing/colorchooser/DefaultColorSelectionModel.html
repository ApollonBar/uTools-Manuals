        public void setSelectedColor(Color color)
        将所选颜色设置为color 。 请注意，将颜色设置为null是未定义的，可能会产生不可预测的结果。 如果将当前颜色设置为新的非null颜色，则此方法将触发状态更改事件。 如果新颜色与当前颜色相同，则不会触发任何事件。
        Specified by:
