    public interface MenuElement
    任何可以放入菜单的组件都应该实现这个接口。 该接口由MenuSelectionManager用于处理菜单层次结构中的选择和导航。

    • 方法详细信息

      • processMouseEvent

        void processMouseEvent(MouseEvent event,
                       MenuElement[] path,
                       MenuSelectionManager manager)
        处理鼠标事件。 event是一个MouseEvent ，源是接收元素的组件。 path是菜单层级中的接收元件的路径，包括接收元件本身。 manager为MenuSelectionManager的菜单层级。 这个方法应该处理MouseEvent ，如果需要，可以通过使用MenuSelectionManager的API来更改菜单选项注意：您不必将事件转发到子组件。 这是由MenuSelectionManager 。

      • menuSelectionChanged

        void menuSelectionChanged(boolean isIncluded)
