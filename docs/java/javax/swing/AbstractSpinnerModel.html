    public abstract class AbstractSpinnerModel
extends Object
implements SpinnerModel, Serializable
    此类提供了SpinnerModel接口的ChangeListener部分，该接口应适用于大多数具体的SpinnerModel实现。 子类必须提供的一个实现setValue ， getValue ， getNextValue和getPreviousValue方法。
    从以下版本开始：
    1.4
    另请参见：
    JSpinner ， SpinnerModel ， SpinnerListModel ， SpinnerNumberModel ， SpinnerDateModel
