    Enclosing class:
    RowFilter < M ， I > 

    public abstract static class RowFilter.Entry<M,I>
extends Object
    Entry对象传递给RowFilter的RowFilter ，允许过滤器获取条目数据的值，从而确定条目是否应该显示。 Entry对象包含有关模型的信息以及从模型获取基础值的方法。
    从以下版本开始：
    1.6
    另请参见：
    RowFilter ， DefaultRowSorter.setRowFilter(javax.swing.RowFilter)
