Interface SpinnerModel

  • 所有已知实现类：
    AbstractSpinnerModel ， SpinnerDateModel ， SpinnerListModel ， SpinnerNumberModel 

    public interface SpinnerModel
    一个可能无限制的对象值序列的模型。 这个模式类似于ListModel但是有一些重要的区别：
    • 序列元素的数量不一定是有限的。
    • 该模型不支持对序列元素的索引随机访问。 一次只能访问三个序列值：当前，下一个和上一个。
    • 当前序列元素可以设置。

    A SpinnerModel有三个属性，只有第一个是读/写。

    value
    序列的当前元素
    nextValue
    以下元素或null如果value是序列的最后一个元素。
    previousValue
    前面的元素或null如果value是序列的第一个元素。
    当value属性更改时，通知ChangeListeners 。 SpinnerModel可以选择通知ChangeListeners下的其他情形。
    从以下版本开始：
    1.4
    另请参见：
    JSpinner ， AbstractSpinnerModel ， SpinnerListModel ， SpinnerNumberModel ， SpinnerDateModel
