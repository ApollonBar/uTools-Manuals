    public class SynthTabbedPaneUI
extends BasicTabbedPaneUI
implements PropertyChangeListener, SynthUI
    为JTabbedPane提供Synth L＆F UI代表。

    从样式查找selectedTabPadInsets属性，其中代表所选选项卡的其他插入。

    从以下版本开始：
    1.7
