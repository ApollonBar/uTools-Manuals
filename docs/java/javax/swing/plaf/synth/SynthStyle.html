    public abstract class SynthStyle
extends Object
    SynthStyle是一套风格属性。 每个SynthUI引用的至少一个SynthStyle被使用获得SynthStyleFactory 。 您通常不需要直接与此类进行交互，而是将Synth File Format file加载到SynthLookAndFeel ，这将创建一组SynthStyles。
    从以下版本开始：
    1.5
    另请参见：
    SynthLookAndFeel ， SynthStyleFactory
