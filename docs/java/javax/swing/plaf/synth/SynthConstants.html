    public interface SynthConstants
    Synth使用的常数。 并非所有组件都支持所有状态。 组件至少将处于主状态之一。 即，从返回值SynthContext.getComponentState()将至少之一ENABLED ， MOUSE_OVER ， PRESSED或DISABLED ，并且还可以包含FOCUSED ， SELECTED或DEFAULT 。
    从以下版本开始：
    1.5
