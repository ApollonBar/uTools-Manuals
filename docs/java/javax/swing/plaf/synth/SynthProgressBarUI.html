        protected Rectangle getBox(Rectangle r)
        存储将在r为当前动画索引绘制的弹跳框的位置和大小，并返回r 。 添加到此类实现paintIndeterminate中绘制的绘图的paintIndeterminate类（例如，在弹跳框中绘制轮廓）可以使用此方法来获取刚被绘制的弹跳框的位置。 通过覆盖此方法，您可以完全控制弹跳框的大小和位置，而无需重新实现paintIndeterminate 。
