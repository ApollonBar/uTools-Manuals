javax.swing.plaf.synth

Class SynthPainter

  • java.lang.Object
    • javax.swing.plaf.synth.SynthPainter

  • public abstract class SynthPainter
extends Object
    SynthPainter用于绘制JComponent的部分。 至少每个JComponent有两种绘画方法：一种用于边框，一种用于背景。 一些JComponent有多个Region ，并因此更多的油漆方法。

    SynthPainter的SynthPainter是从SynthStyle.getPainter(javax.swing.plaf.synth.SynthContext)方法获得的。

    您通常提供一个SynthPainter由合成器的方式file格式。 以下示例为所有JButton注册了一个绘画器，将渲染图像myImage.png ： 

      <style id="buttonStyle">
    <imagePainter path="myImage.png" sourceInsets="2 2 2 2"
                  paintCenter="true" stretch="true"/>
    <insets top="2" bottom="2" left="2" right="2"/>
  </style>
  <bind style="buttonStyle" type="REGION" key="button"/>

    SynthPainter是抽象的，只要它没有绘画，所有的方法都是空的。 虽然这些方法都没有类型来引发异常，但子类可以假设有效参数被传入，如果不是，它们可以抛出一个NullPointerException或IllegalArgumentException来响应无效参数。
