      • drawLine

        public void drawLine(SynthContext context,
                     Object paintKey,
                     Graphics g,
                     int x1,
                     int y1,
                     int x2,
                     int y2,
                     Object styleKey)
        在两个终点之间画一条线。

        此实现仅支持一种线型键"dashed" 。 "dashed"线条样式仅适用于垂直和水平线。

        指定null从任意键或不同"dashed"将绘制实线。

        参数
        context - 标识托管区域
        paintKey - 标识要求绘制的组件的部分，可以为null
        g - 要绘制的图形对象
        x1 - x起源
        y1 - y来源
        x2 - x目的地
        y2 - y目的地
        styleKey - 识别所请求的行的样式（例如“虚线”）
        从以下版本开始：
        1.6
