    MouseWheelHandler是一个实现MouseWheelListener接口的内部类。 MouseWheelHandler通过相应地滚动JScrollPane来响应MouseWheelEvents。 如果滚动窗格的isWheelScrollingEnabled方法返回false，则不会发生滚动。
    从以下版本开始：
    1.4
    另请参见：
    JScrollPane.isWheelScrollingEnabled() ， BasicScrollPaneUI.createMouseWheelListener() ， MouseWheelListener ， MouseWheelEvent
