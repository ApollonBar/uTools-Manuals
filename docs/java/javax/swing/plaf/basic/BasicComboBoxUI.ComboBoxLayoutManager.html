    该布局管理器处理组合框的“标准”布局。 它将箭头按钮向右，编辑器向左。 如果没有编辑器，它仍然保持箭头按钮在右边。 这个公共内部类应该被视为受保护的。 仅在BasicComboBoxUI的BasicComboBoxUI类中BasicComboBoxUI 。
