implements PropertyChangeListener
    在InstallUI时添加到JList的PropertyChangeListener。 当影响布局的JList属性的值发生变化时，我们在updateLayoutStateNeeded中设置一点。 如果JLists模型发生变化，我们还会从旧模型中删除我们的听众。 同样对于JList selectionModel。

    警告：此类的序列化对象与将来的Swing版本不兼容。 当前的序列化支持适用于运行相同版本的Swing的应用程序之间的短期存储或RMI。 从1.4开始，对所有JavaBeans的长期存储的支持已经添加到java.beans包中。 请参阅XMLEncoder 。

    另请参见：
    BasicListUI.maybeUpdateLayoutState() ， BasicListUI.createPropertyChangeListener() ， BasicListUI.installUI(javax.swing.JComponent)
