    在installUI时添加到JLists选择模型的ListSelectionListener，以及JList.selectionModel属性更改时。 当选择更改时，我们重新绘制受影响的行。

    警告：此类的序列化对象与将来的Swing版本不兼容。 当前的序列化支持适用于运行相同版本的Swing的应用程序之间的短期存储或RMI。 从1.4版本开始，所有JavaBeans的长期存储支持已被添加到java.beans软件包中。 请参阅XMLEncoder 。

    另请参见：
    BasicListUI.createListSelectionListener() ， BasicListUI.getCellBounds(javax.swing.JList, int, int) ， BasicListUI.installUI(javax.swing.JComponent)
