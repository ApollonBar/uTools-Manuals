        public static void drawString(Graphics g,
                              String text,
                              int underlinedChar,
                              int x,
                              int y)
        绘制一个字符串与图形g在位置（x，y）就像g.drawString会。 文字中第一次出现underlineChar将被强调。 匹配算法不区分大小写。

      • drawStringUnderlineCharAt

        public static void drawStringUnderlineCharAt(Graphics g,
                                             String text,
                                             int underlinedIndex,
                                             int x,
                                             int y)
        画一个字符串与图形g在位置（ x ， y ）就像g.drawString会。 索引号为underlinedIndex的文字中的字符将被加下划线。 如果index超出了text （包括<0）的范围，则不会强调任何内容。
        参数
        g - 图形绘制
        text - 要绘制的字符串
        underlinedIndex - 文本中下划线的字符索引
        x - x坐标绘制
        y - y坐标绘制
        从以下版本开始：
        1.4
