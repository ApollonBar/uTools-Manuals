Class BasicHTML

  • java.lang.Object
    • javax.swing.plaf.basic.BasicHTML

  • public class BasicHTML
extends Object
    支持为swing组件提供html视图。 这将一个简单的html字符串转换为可以渲染html并提供必要的布局语义的javax.swing.text.View实现。
    从以下版本开始：
    1.3

      • updateRenderer

        public static void updateRenderer(JComponent c,
                                  String text)
        将给定文本的HTML渲染存储到给定JComponent的客户端属性中。 如果给定的文本不是HTML，则该属性将被清除任何渲染器。

        该方法对于静态（即共享）并且完全从JComponent获取状态的ComponentUI实现是有用的。
