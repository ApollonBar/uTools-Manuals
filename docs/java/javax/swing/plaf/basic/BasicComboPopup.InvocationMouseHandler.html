    在组合框（ 而不是其弹出菜单）上注册的侦听器来处理影响弹出菜单状态的鼠标事件。 这个听众的主要目的是使弹出菜单出现并消失。 如果鼠标在拖动期间在列表中被释放，则此侦听器还可以通过设置选择来帮助点击和拖动方案。

    警告：我们建议您不要创建此类的子类。 如果您绝对必须创建一个子类，请确保调用每个方法的超类版本。
