    public static class BasicOptionPaneUI.ButtonAreaLayout

implements LayoutManager
    ButtonAreaLayout行为方式与FlowLayout类似。 它从左到右列出了所有组件。 如果syncAllWidths为真，则每个组件的宽度将被设置为最大的优选尺寸宽度。 这个类应该被视为一个“受保护”的内部类。 仅在BasicOptionPaneUI的BasicOptionPaneUI类中BasicOptionPaneUI 。
