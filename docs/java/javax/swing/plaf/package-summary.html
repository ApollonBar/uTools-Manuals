Package javax.swing.plaf Description

提供一个接口和许多抽象类，Swing用来提供其可插拔的外观和感觉功能。 它的类被子类化，并通过外观和UI实现，如Basic和Java外观（Metal）。 此套件仅由不能通过对现有外观组件（如javax.swing.plaf.basic和javax.swing.plaf.metal软件包提供的组件）进行javax.swing.plaf.basic类化创建新外观的开发人员使用。

注意：大多数Swing API 不是线程安全的。 有关详细信息，请参阅Threads and Swing ，在部分The Java Tutorial 。
