      • getIndex

        public int getIndex()
        返回列表中放置数据的索引。 值的解释取决于相关组件上设置的放置模式。 如果下拉模式是DropMode.USE_SELECTION或DropMode.ON ，则返回值是列表中行的索引。 如果drop模式为DropMode.INSERT ，则返回值是DropMode.INSERT插入数据的索引。 如果放置模式是DropMode.ON_OR_INSERT ，的值isInsert()表示索引是否是行的索引，或插入索引。

        -1表示下降发生在空白空间，无法计算索引。

        结果
        下降指数
