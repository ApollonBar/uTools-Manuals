    public abstract class InternalFrameAdapter

    用于接收内部帧事件的抽象适配器类。 这个类中的方法是空的。 该类作为创建侦听器对象的方便而存在，在功能上等同于AWT中的WindowAdapter类。

    请参阅Java Tutorial中的 How to Write an Internal Frame Listener
