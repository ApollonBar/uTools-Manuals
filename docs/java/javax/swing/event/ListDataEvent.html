    public class ListDataEvent
extends EventObject
    定义将更改封装到列表的事件。

    警告：此类的序列化对象与将来的Swing版本不兼容。 当前的序列化支持适用于运行相同版本的Swing的应用程序之间的短期存储或RMI。 从1.4开始，对所有JavaBeans的长期存储的支持已经添加到java.beans包中。 请参阅XMLEncoder 。

                     int type,

        构造一个ListDataEvent对象。 如果index0是> index1，则index0和index1将被交换，使得index0始终为<= index1。
        参数
        source - 源对象（通常为 this ）
        type -一个int，指定 CONTENTS_CHANGED ， INTERVAL_ADDED ，或 INTERVAL_REMOVED

        index1 - 新间隔的另一端
