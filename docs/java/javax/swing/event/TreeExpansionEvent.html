    public class TreeExpansionEvent
extends EventObject
    用于标识树中单个路径的事件。 getSource返回的源将是JTree的一个实例。

    有关进一步的文档和示例，请参阅Java教程中的以下部分： How to Write a Tree Expansion Listener和How to Write a Tree-Will-Expand Listener 。

    警告：此类的序列化对象与将来的Swing版本不兼容。 当前的序列化支持适用于运行相同版本的Swing的应用程序之间的短期存储或RMI。 从1.4开始，对所有JavaBeans的长期存储的支持已经添加到java.beans包中。 请参阅XMLEncoder 。
