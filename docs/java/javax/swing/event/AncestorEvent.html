    public class AncestorEvent
extends AWTEvent
    向组件层级中的祖先发送的子组件的事件。

    警告：此类的序列化对象与将来的Swing版本不兼容。 当前的序列化支持适用于运行相同版本的Swing的应用程序之间的短期存储或RMI。 从1.4版本开始，支持所有JavaBeans的长期存储已经添加到java.beans包中。 请参阅XMLEncoder 。

      • AncestorEvent

        public AncestorEvent(JComponent source,
                     int id,
                     Container ancestor,
                     Container ancestorParent)
        构造一个AncestorEvent对象，以识别祖先组件的显示状态的变化。
        参数
        source - 发起事件的JComponent（通常为 this ）
        id -一个int，指定 ANCESTOR_ADDED ， ANCESTOR_REMOVED或 ANCESTOR_MOVED
        ancestor - 指定显示状态更改的祖先组件的Container对象
        ancestorParent - 指定祖先父项的Container对象
