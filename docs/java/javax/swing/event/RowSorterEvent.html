    RowSorterEvent提供了对RowSorter的更改RowSorter 。 有两种类型的通知是可能的：
    • Type.SORT_ORDER_CHANGED ：表示排序顺序已更改。 这通常是以下通知：
    • Type.SORTED ：表示模型的内容已经以某种方式进行了转换。 例如，内容可能已被排序或过滤。
