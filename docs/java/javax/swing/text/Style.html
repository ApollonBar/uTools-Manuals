    public interface Style
extends MutableAttributeSet
    与文档中元素关联的属性集合。 由于这些通常用于将字符和段落样式与元素相关联，因此提供了此操作。 与元素相关联的其他自定义属性将有效地位于层次结构中的名称 - 值对，如果在本地未找到名称（密钥），则请求将转发给父级。 通常使用的属性被分离出来，以便于更有效的替代实现。
