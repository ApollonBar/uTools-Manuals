    public class FieldView
extends PlainView
    将多行纯文本视图扩展为适合单行编辑器视图。 如果视图被分配了额外的空间，则该字段必须进行调整。 如果托管组件是JTextField，则该视图将管理相关联的BoundedRangeModel的范围，并调整水平分配以匹配JTextField的当前可见性设置。
