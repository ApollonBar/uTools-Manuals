    public abstract class AbstractDocument.AbstractElement
extends Object
implements Element, MutableAttributeSet, Serializable, TreeNode
    实现元素的抽象部分。 默认情况下，元素通过具有表示元素当前属性集的不可变部分的字段来支持属性。 元素本身实现MutableAttributeSet，可以通过获取新的不可变集来修改集合。 不可变集由与文档相关联的AttributeContext提供。

    警告：此类的序列化对象与将来的Swing版本不兼容。 当前的序列化支持适用于运行相同版本的Swing的应用程序之间的短期存储或RMI。 从1.4开始，对所有JavaBeans的长期存储的支持已经添加到java.beans包中。 请参阅XMLEncoder 。
