    public class StyledEditorKit
extends DefaultEditorKit
    这是文本组件所需的一组事物，作为某种类型的文本文档的合理功能的编辑器。 此实现提供了将文本视为样式文本的默认实现，并提供了编辑样式文本的最小操作集。
