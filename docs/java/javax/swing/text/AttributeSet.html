    public interface AttributeSet
    一组独特的属性。 这是一个只读，不可变的界面。 属性基本上是一个键和分配给键的值。 该集合可以表示类似于风格运行，逻辑风格等。这些通常用于描述将有助于一些图形表示（诸如字体）的特征。 一组可能的键是无限的，可以是任何东西。 通常，View实现将响应属性定义并呈现某些东西来表示属性。

    属性可以在层次结构中潜在地解决。 如果密钥本地不解析，并且存在解析父级，则密钥将通过父级解析。
