    AsyncBoxView 

    public class AsyncBoxView.ChildLocator
extends Object
    在本地化区域内进行变更时，可以管理子视图在局部区域中的有效位置。 AsyncBoxView可能会不断更改，但可见区域需要保持相当稳定，直到布局线程决定向父级发布更新。
    从以下版本开始：
    1.3

        将负责偏移的视图定位到沿主轴的框中。 确保在直到给定目标跨越所需偏移量的ChildState对象上设置偏移量。
