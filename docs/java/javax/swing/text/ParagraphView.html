    public class ParagraphView
extends FlowView
implements TabExpander
    查看一个简单的换行段落，支持多种字体，颜色，组件，图标等。它基本上是一个带有边缘的垂直框。 盒子的内容是一排是特殊的水平盒子。 此视图创建一个视图集合，代表段落元素的子元素。 将这些视图中的每一个直接放入行中，否则将breakView方法来尝试将视图划分成适合的部分。
