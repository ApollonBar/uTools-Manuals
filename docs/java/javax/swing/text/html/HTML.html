        public static HTML.Tag getTag(String tagName)
        获取一个公知标签名称的标签常量（即集合{A，ADDRESS，APPLET，AREA，B，BASE，BASEFONT，BIG，BLOCKQUOTE，BODY，BR，CAPTION，CENTER，CITE，CODE中的一个标签，DD，DFN，DIR，DIV，DL，DT，EM，FONT，FORM，FRAME，FRAMESET，H1，H2，H3，H4，H5，H6，HEAD，HR，HTML，I，IMG，INPUT，ISINDEX，KBD ，LI，LINK，MAP，MENU，META，NOBR，NOFRAMES，OBJECT，OL，OPTION，P，PARAM，PRE，SAMP，SCRIPT，SELECT，SMALL，SPAN，STRIKE，S，STRONG，STYLE，SUB，SUP，TABLE ，TD，TEXTAREA，TH，TITLE，TR，TT，U，UL，VAR}。如果给定的名称不代表其中一个众所周知的标签，那么将返回 null 。
        参数
        tagName - 要求的 String名称
        结果
        一个对应于 tagName的标签常量，或 null如果没有找到
