      • mouseDragged

        public void mouseDragged(MouseEvent e)
        描述从类别复制： MouseAdapter
        在组件上按下鼠标按钮然后拖动时调用。 MOUSE_DRAGGED事件将继续传递到拖动发起的组件，直到鼠标按钮释放（无论鼠标位置是否在组件的边界内）。

        由于平台相关的拖放实现， MOUSE_DRAGGED事件可能不会在本地拖放操作期间传递。
