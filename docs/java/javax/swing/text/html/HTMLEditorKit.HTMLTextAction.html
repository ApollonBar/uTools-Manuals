extends StyledEditorKit.StyledTextAction
    一个抽象的Action提供了一些方便的方法，可能有助于将HTML插入到现有文档中。

    注意：没有方便的方法获取文档上的锁定。 如果您有另一个线程修改文本，这些方法可能会有不一致的行为，或返回错误的东西。
