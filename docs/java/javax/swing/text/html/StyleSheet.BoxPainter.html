        public float getInset(int side,
                      View v)
        获取给定方面所需的插图，以说明边距，边框和填充。
        参数
        side - 获取插图的框的大小。 这可以是View.TOP，View.LEFT，View.BOTTOM或View.RIGHT。
        v - 提出请求的视图。 这用于获取AttributeSet，可用于解析百分比参数。
        异常
        IllegalArgumentException - 无效方向
