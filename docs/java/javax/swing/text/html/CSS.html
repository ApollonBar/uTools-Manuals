    public class CSS

    定义一组CSS attributes作为类型安全枚举。 HTML View实现使用CSS属性来确定它们将如何呈现。 这也定义了CSS / HTML / StyleConstants之间的映射方法。 任何速记属性（如字体）都映射到内在属性。
