    public final class AttributeList
extends Object
implements DTDConstants, Serializable
    该类使用ATTLIST构造在DTD中描述了SGML元素的属性。 AttributeList可以使用getAttributes（）方法从Element类中获取。

    它实际上是链表中的一个元素。 重复使用getNext（）方法枚举元素的所有属性。
