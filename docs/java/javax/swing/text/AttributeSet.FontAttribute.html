    public static interface AttributeSet.FontAttribute
    此接口是预期存在于任何有助于确定用于呈现某些文本的字体的任何属性键上的类型签名。 这不被认为是封闭的集合，该定义可以跨平台的版本进行更改，并且可以通过对应于特定于某种类型的内容的逻辑设置的附加用户添加的条目进行修改。
