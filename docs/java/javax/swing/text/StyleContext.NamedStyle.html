    public class StyleContext.NamedStyle
extends Object
implements Style, Serializable
    属性集合，通常用于表示字符和段落样式。 这是MutableAttributeSet的一个实现，如果需要可以观察。 这些风格将利用不变性，而集合足够小，并且可能比SimpleAttributeSet更有效。

    警告：此类的序列化对象与将来的Swing版本不兼容。 当前的序列化支持适用于运行相同版本的Swing的应用程序之间的短期存储或RMI。 从1.4开始，对所有JavaBeans的长期存储的支持已经添加到java.beans包中。 请参阅XMLEncoder 。
