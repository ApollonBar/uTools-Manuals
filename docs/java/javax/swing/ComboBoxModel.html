    public interface ComboBoxModel<E>
extends ListModel<E>
    组合框的数据模型。 此界面扩展了ListDataModel ，并添加了所选项目的概念。 所选项目通常是组合框显示区域中可见的项目。

    所选项目可能不一定由底层的ListModel管理。 这种不相交的行为允许在模型中临时存储和检索所选择的项目。
