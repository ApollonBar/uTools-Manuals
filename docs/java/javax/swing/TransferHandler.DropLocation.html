    表示应插入丢弃数据的位置。 这是一个仅封装一个点的基类。 支持drop的组件可以提供包含更多信息的子类。

    开发人员通常不应该创建或扩展这个类的实例。 相反，这些是由DnD实现由TransferSupport实例和具有getDropLocation()方法的组件提供的。

    从以下版本开始：
    1.6
