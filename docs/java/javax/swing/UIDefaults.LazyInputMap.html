    LazyInputMap将在其createValue方法中创建一个InputMap 。 绑定在构造函数中传入。 绑定是一个数组，偶数条目为字符串KeyStrokes （例如“alt SPACE”），奇数条目是InputMap中使用的值（以及ActionMap中的ActionMap ）。
