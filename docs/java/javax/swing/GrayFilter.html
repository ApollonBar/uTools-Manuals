        public GrayFilter(boolean b,
                  int p)
        构造一个将彩色图像过滤到灰度图像的GrayFilter对象。 按钮用于创建禁用（“灰色”）按钮图像。
        参数
        b - 一个布尔值 - 如果像素应该变亮，则为真
        p - 一个在0..100范围内的int，它决定了灰色的百分比，其中100是最深的灰色，0是最亮的
