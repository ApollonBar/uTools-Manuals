    public interface ListModel<E>
    该界面定义了JList使用的方法组件来获取列表中每个单元格的值以及列表的长度。 逻辑上，模型是向量，索引从0到ListDataModel.getSize（） - 1。对数据模型的内容或长度的任何更改都必须报告给所有的ListDataListeners。
