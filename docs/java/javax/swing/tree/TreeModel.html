Interface TreeModel

  • 所有已知实现类：
    DefaultTreeModel 

    public interface TreeModel
    该模型由JTree 。

    JTree及其相关类广泛使用的TreePath S表示在确定节点TreeModel 。 如果TreeModel返回相同的对象，与equals相比，在相同父母的两个不同指数下，所得到的TreePath对象也将被视为相等。 一些实现可以假设如果两个TreePath相同，则它们标识相同的节点。 如果不符合条件，可能会导致绘画问题和其他怪物。 换句话说，如果给定父级的getChild返回相同的对象（由equals ），可能会导致问题，建议您避免这样做。

    同样JTree及其相关课程在TreePath s中Map s。 因此，如果节点被请求两次，则返回值必须相等（使用equals方法）并具有相同的hashCode 。

    有关树模型的更多信息，包括自定义实现的示例，请参阅Java教程中的 How to Use Trees 。
