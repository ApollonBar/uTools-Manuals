      • getNodeDimensions

        public abstract Rectangle getNodeDimensions(Object value,
                                            int row,
                                            int depth,
                                            boolean expanded,
                                            Rectangle bounds)
        通过边界的引用返回大小和x的起始位置。 调用方法负责确定Y位置。 如果边界为null ，则应返回新创建的Rectangle ，否则该值应放置在边界并返回。
        参数
        value -的 value被表示
        row - 正在查询的行
        depth - 行的深度
        expanded - 如果行被展开，则为true，否则为false
        bounds - a Rectangle包含所需的大小，代表 value
        结果
        一个 Rectangle包含节点的尺寸，或 null如果节点没有尺寸
