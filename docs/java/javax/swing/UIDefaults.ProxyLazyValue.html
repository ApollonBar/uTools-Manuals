implements UIDefaults.LazyValue
    该类提供了一个LazyValue的LazyValue ，可以用于延迟加载要创建的实例的Class。 它也避免为LazyValue子类创建一个匿名内部类。 这两项改善，机管局观感加载，在一个很小的性能下降的第一次成本的实时性能createValue被称为（因为反射API的使用）。
    从以下版本开始：
    1.3
