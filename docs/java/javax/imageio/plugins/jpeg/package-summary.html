Package javax.imageio.plugins.jpeg Description

支持内置JPEG插件的类。

此软件包包含一些内置JPEG读写器插件的支持类。 提供了代表量化和Huffman表类，以及扩展ImageReadParam和ImageWriteParam提供了在读写过程中提供的表。 有关内置JPEG插件的操作的更多信息，请参阅JPEG metadata format specification and usage notes 。
