    • 构造方法详细信息

      • FileCacheImageInputStream

        public FileCacheImageInputStream(InputStream stream,
                                 File cacheDir)
                          throws IOException
        构造一个FileCacheImageInputStream ，它将从给定的InputStream读取。

        临时文件用作缓存。 如果cacheDir是非null并且是一个目录，该文件将在那里创建。 如果是null ，将使用系统相关的默认临时文件目录（有关详细信息，请参阅File.createTempFile的File.createTempFile ）。

        参数
        stream - 要阅读的 InputStream 。
        cacheDir - 一个 File指示应该创建高速缓存文件的位置，或 null使用系统目录。
        异常
        IllegalArgumentException - 如果 stream是 null 。
        IllegalArgumentException - 如果 cacheDir是非 null但不是一个目录。
        IOException - 如果无法创建缓存文件。
