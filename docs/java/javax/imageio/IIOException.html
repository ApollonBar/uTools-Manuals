    public class IIOException
extends IOException
    用于发信号读取和写入操作失败的异常类。

    除了消息字符串之外，还保留对另一个Throwable （ Error或异常 ）的引用。 这个引用，如果null ，是引发这个异常发生的事件。 例如，从IOException读取的File将被存储在那里。
