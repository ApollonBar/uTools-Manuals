    Serializable 

    public class ActivityCompletedException
extends RemoteException
    可能会在访问活动上下文的任何方法上抛出此异常，并指示活动中正在进行的工作是不可能的。 这可能是因为已经通过CompletionStatusFailOnly指示完成活动，或者由于超时而结束。 此异常将通过org.omg.CORBA.ACTIVITY_COMPLETED系统异常通过ORB边界传播。 应用程序应通过尝试完成活动来处理此错误。
