    Serializable 

    public class InvalidActivityException
extends RemoteException
    可以在访问Activity上下文的任何方法上抛出此异常，并指示与尝试调用相关联的尝试调用或活动上下文与Activity的当前状态不兼容。 如果在禁止Activity上下文的方法上接收到Activity上下文，则它也可能被容器抛出。 该异常将通过org.omg.CORBA.INVALID_ACTIVITY系统异常跨越J2EE活动服务ORB边界传播。 应用程序应通过尝试完成活动来处理此错误。
