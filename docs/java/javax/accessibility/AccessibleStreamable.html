javax.accessibility

Interface AccessibleStreamable


  • public interface AccessibleStreamable
    所述AccessibleStreamable接口应该由实施AccessibleContext呈现在显示屏幕上的部件的后面的原始流的任何组分。 这些组件的示例是HTML，位图图像和MathML。 实现AccessibleStreamable的对象提供两件事情：对象支持的MIME类型列表以及每个MIME类型的流接口以获取数据。
    从以下版本开始：
    1.5
    另请参见：
    AccessibleContext
