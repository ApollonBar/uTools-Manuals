    public interface AccessibleEditableText
extends AccessibleText

    AccessibleEditableText接口应该由显示器上显示可编辑文本信息的所有类来实现。 与AccessibleText接口一起，该接口提供辅助技术通过其内容，属性和空间位置访问该文本的标准机制。 应用程序可以通过首先获取其AccessibleContext（参见Accessible ），然后调用AccessibleContext的AccessibleContext.getAccessibleEditableText()方法来确定对象是否支持AccessibleEditableText接口。 如果返回值不为空，则该对象支持该接口。

    从以下版本开始：
    1.4
    另请参见：
    Accessible ， Accessible.getAccessibleContext() ， AccessibleContext ， AccessibleContext.getAccessibleText() ， AccessibleContext.getAccessibleEditableText()
