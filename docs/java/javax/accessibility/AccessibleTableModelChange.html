Interface AccessibleTableModelChange

  • 所有已知实现类：
    JTable.AccessibleJTable.AccessibleJTableModelChange 

    public interface AccessibleTableModelChange
    AccessibleTableModelChange接口描述对表模型的更改。 模型变化的属性可以通过以下方法获得：
    • public int getType（）
    • public int getFirstRow（）;
    • public int getLastRow（）;
    • 公共int getFirstColumn（）;
    • public int getLastColumn（）;
    getType（）返回的模型更改类型将是以下之一：
    • INSERT - 插入了一个或多个行和/或列
    • UPDATE - 某些表数据已更改
    • 删除 - 一行或多行和/或列已被删除
    表的受影响区域可以由指定行和列范围的其他四种方法确定
    从以下版本开始：
    1.3
    另请参见：
    Accessible ， Accessible.getAccessibleContext() ， AccessibleContext ， AccessibleContext.getAccessibleTable()
