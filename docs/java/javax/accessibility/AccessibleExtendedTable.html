    public interface AccessibleExtendedTable
extends AccessibleTable
    Class AccessibleExtendedTable提供有关以二维表格格式显示数据的用户界面组件的扩展信息。 应用程序可以通过首先获取其AccessibleContext，然后调用AccessibleContext.getAccessibleTable()方法来确定对象是否支持AccessibleExtendedTable接口。 如果返回值不为null，返回值的类型为AccessibleExtendedTable，则该对象支持该接口。

    1.4

      • getAccessibleRow

        int getAccessibleRow(int index)
        返回表中索引的行号。
        参数
        index - 表中的从零开始的索引。 索引是从行== 0和列== 0的表单元格偏移量。
        结果
        如果存在，则表的基于零的行; 否则-1。
