Interface AccessibleIcon

  • 所有已知实现类：
    ImageIcon.AccessibleImageIcon 

    public interface AccessibleIcon
    AccessibleIcon接口应该由具有关联图标的任何对象（例如，按钮）支持。 该界面提供辅助技术的标准机制，以获取有关图标的描述性信息。 应用程序可以通过首先获取其AccessibleContext（参见Accessible ），然后调用AccessibleContext.getAccessibleIcon()方法来确定对象是否支持AccessibleIcon接口。 如果返回值不为空，则该对象支持该接口。
    从以下版本开始：
    1.3

    Accessible ， AccessibleContext
