    public abstract class AccessibleBundle
extends Object

    用于维护强类型枚举的基类。 这是AccessibleState和AccessibleRole的超类 。

    toDisplayString方法允许您从此类中定义的键从预定义的ResourceBundle中获取与区域设置无关的键的本地化字符串。 这个本地化的字符串旨在被人类所读取。

      • toDisplayString

        protected String toDisplayString(String resourceBundleName,
                                 Locale locale)
        获取密钥作为本地化字符串。 如果找不到密钥的本地化字符串，将返回存储在角色中的区域设置独立密钥。 此方法仅用于子类，以便它们可以指定其自己的资源束，其中包含其密钥的本地化字符串。
        参数
        resourceBundleName - 用于查找的资源束的名称
        locale - 要获取本地化字符串的区域设置
        结果
        一个本地化的字符串的键。
