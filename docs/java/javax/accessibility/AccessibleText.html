    public interface AccessibleText

    AccessibleText界面应由显示文本信息的所有类实现。 该界面提供了辅助技术通过其内容，属性和空间位置访问该文本的标准机制。 应用程序可以通过首先获取AccessibleContext（参见Accessible ），然后调用AccessibleContext的AccessibleContext.getAccessibleText()方法来确定对象是否支持AccessibleText接口。 如果返回值不为空，则该对象支持该接口。

    另请参见：
    Accessible ， Accessible.getAccessibleContext() ， AccessibleContext ， AccessibleContext.getAccessibleText()

      • SENTENCE

        static final int SENTENCE
        常数用于表示应检索的部分文本是一个句子。 一句话是一串表达一个断言，一个问题，一个命令，一个愿望，一个感叹号或一个动作的表现的单词。 在英文语言中，字符串通常以大写字母开头，并以适当的结尾标点符号结尾; 例如期间，问题或感叹号。 其他地区可能会使用不同的大小写和/或标点符号。
        另请参见：
        getAtIndex(int, int) ， getAfterIndex(int, int) ， getBeforeIndex(int, int) ， Constant Field Values

    • 方法详细信息

      • getIndexAtPoint

        int getIndexAtPoint(Point p)
        给定本地坐标点，返回该点下的字符从零开始的索引。 如果该点无效，此方法返回-1。
        参数
        p - 本地坐标点
        结果
        Point p处字符的零基索引; 如果Point无效返回-1。

      • getCharacterBounds

        Rectangle getCharacterBounds(int i)
        将给定索引处的字符的边框确定为字符串。 边界以局部坐标返回。 如果索引无效，则返回空矩形。
        参数
        i - 索引到String中
        结果
        字符的边框的屏幕坐标，如果索引无效返回一个空矩形。
