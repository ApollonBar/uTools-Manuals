javax.accessibility

Interface AccessibleKeyBinding


  • public interface AccessibleKeyBinding
    任何具有诸如键盘助记符和/或键盘快捷键之类的键盘绑定的对象都可以支持AccessibleKeyBinding接口，可用于选择对象。 该界面提供了辅助技术的标准机制，以确定此对象存在的键绑定。 任何具有这种密钥绑定的对象都应该支持该接口。
    从以下版本开始：
    1.4
    另请参见：
    Accessible ， Accessible.getAccessibleContext() ， AccessibleContext
