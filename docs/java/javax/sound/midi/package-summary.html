Package javax.sound.midi

提供MIDI（乐器数字接口）数据的I / O，排序和综合的接口和类。

See: 描述

  • 接口摘要  
    接口 描述
    ControllerEventListener
    该 ControllerEventListener接口应由其实例需要在通知中实现 Sequencer已经处理MIDI控制更改事件的请求类型。
    MetaEventListener
    该 MetaEventListener接口应由其实例需要在通知中实现 Sequencer处理了 MetaMessage 。
    MidiChannel
    MidiChannel对象表示单个MIDI通道。
    MidiDevice
    MidiDevice是所有MIDI设备的基础接口。
    MidiDeviceReceiver
    MidiDeviceReceiver是一个 Receiver ，它代表一个MidiDevice的MIDI输入连接 MidiDevice （见 MidiDevice.getReceiver() ）。
    MidiDeviceTransmitter
    MidiDeviceTransmitter是一个 Transmitter ，代表一个MidiDevice的MIDI输入连接 MidiDevice （见 MidiDevice.getTransmitter() ）。
    Receiver
    A Receiver接收到 MidiEvent对象，通常会做出有用的响应，例如解释它们生成声音或原始MIDI输出。
    Sequencer
    该回放MIDI硬件或软件设备 sequence称为 序 。
    Soundbank
    一个 Soundbank包含一组 Instruments可以加载到 Synthesizer 。
    Synthesizer
    A Synthesizer产生声音。
    Transmitter
    A Transmitter向一个或多个 Receivers发送 MidiEvent对象。
  • 类摘要  
    类 描述
    Instrument
    仪器是具有某些参数设置的声音合成算法，通常设计为模拟特定的真实乐器或实现特定类型的声音效果。
    MetaMessage
    A MetaMessage是 MidiMessage ，它对合成器无意义，但可以存储在MIDI文件中并由音序器程序解释。
    MidiDevice.Info
    MidiDevice.Info对象包含有关 MidiDevice各种数据，包括其名称，创建它的公司和描述性文本。
    MidiEvent
    MIDI事件包含一个MIDI消息和相应的时间戳记，以秒表形式表示，并且可以表示存储在MIDI文件或 Sequence对象中的MIDI事件信息。
    MidiFileFormat
    一个 MidiFileFormat对象封装了一个MIDI文件的类型，以及它的长度和时间信息。
    MidiMessage
    MidiMessage是MIDI消息的基类。
    MidiSystem
    MidiSystem课程提供对安装的MIDI系统资源的访问，包括诸如合成器，音序器和MIDI输入和输出端口的设备。
    Patch
    Patch对象表示在MIDI合成器上存储（加载）单个乐器的位置。
    Sequence
    甲 Sequence是包含音乐信息可以由回放的数据结构（通常是整首歌曲或组合物） Sequencer对象。
    Sequencer.SyncMode
    一个 SyncMode对象表示MIDI音序器的时间概念可以与主设备或从设备同步的方式之一。
    ShortMessage
    A ShortMessage包含一个MIDI消息，其状态字节后面最多有两个数据字节。
    SoundbankResource
    一个 SoundbankResource表示存储在任何音频资源 Soundbank 。
    SysexMessage
    A SysexMessage对象表示MIDI系统专用消息。
    Track
    MIDI轨道是独立的MIDI事件流（时间戳的MIDI数据），可以与标准MIDI文件中的其他轨道一起存储。
    VoiceStatus
    A VoiceStatus对象包含有关由Synthesizer生成的其中一个语音的当前状态的信息 。
  • 异常摘要  
    异常 描述
    InvalidMidiDataException
    一个 InvalidMidiDataException指示遇到了不恰当的MIDI数据。
    MidiUnavailableException
    一个 MidiUnavailableException当请求的MIDI组件无法打开或创建的，因为它是不可被抛出。
