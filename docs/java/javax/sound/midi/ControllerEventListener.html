        void controlChange(ShortMessage event)
        当Sequencer遇到并处理该侦听器感兴趣的控制更改事件时调用。 传入的事件是一个ShortMessage ，其第一个数据字节指示控制器编号，第二个数据字节是控制器设置的值。
        参数
        event - 顺控程序在其正在处理的序列中遇到的控制更改事件
