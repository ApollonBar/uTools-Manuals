    public class InvalidMidiDataException
extends 异常
    一个InvalidMidiDataException指示遇到了不恰当的MIDI数据。 这通常意味着从MIDI规范的角度看，数据本身是无效的。 一个例子将是一个未定义的状态字节。 但是，异常可能只是意味着数据在使用的上下文中无效，或者数据被给出的对象无法解析或使用它。 例如，即使在MIDI规范中定义了格式，文件读取器可能无法解析Type 2 MIDI文件。
