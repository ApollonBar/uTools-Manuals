    public class MidiUnavailableException
extends 异常
    一个MidiUnavailableException当请求的MIDI组件无法打开或创建的，因为它是不可被抛出。 当设备在另一个应用程序使用时，通常会发生这种情况。 更一般地说，当有一定数量的某种资源可以用于某些目的时，可能会发生这种情况，并且它们都已经在使用（也许全部由本应用程序使用）。 关于后一种情况的一个例子，参见setReceiver方法Transmitter 。
