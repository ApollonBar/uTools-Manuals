  • public abstract class EnumControl
extends Control
    A EnumControl提供对一组离散可能值的控制，每个值由对象表示。 在图形用户界面中，这样的控件可以由一组按钮来表示，每个按钮选择一个值或设置。 例如，混响控制可以提供多个预设的混响设置，而不是提供将由FloatControl对象表示的FloatControl 。

    只提供两个设置之间进行选择控件通常可以代替实现为BooleanControl ，并提供带一些可量化的维度的一组值的控件可能被改为实现为FloatControl用粗的分辨率。 但是， EnumControl一个关键特性是返回的值是任意对象，而不是数值或布尔值。 这意味着每个返回的对象都可以提供更多的信息。 例如， REVERB控件的设置是ReverbType的ReverbType ，可以查询每个设置使用的参数值。
