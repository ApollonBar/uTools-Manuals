    public interface Port
extends Line
    端口是用于输入或输出音频到音频设备的简单线。 作为源线（混频器输入）的端口的常见示例包括麦克风，线路输入和CD-ROM驱动器。 作为目标线（混频器输出）的端口包括扬声器，耳机和线路输出。 您可以使用Port.Info对象访问端口。
