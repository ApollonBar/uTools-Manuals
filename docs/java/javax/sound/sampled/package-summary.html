Package javax.sound.sampled

提供用于捕获，处理和播放采样音频数据的接口和类。

See: 描述

  • 接口摘要  
    接口 描述
    Clip
    Clip接口代表一种特殊类型的数据线，其音频数据可以在播放之前加载，而不是实时流式传输。
    DataLine
    DataLine添加媒体相关功能， Line 。
    Line
    所述 Line接口表示单声道或多声道音频馈送。
    LineListener
    实现LineListener接口的类的 LineListener可以注册以在线路状态更改时接收事件。
    Mixer
    混音器是具有一行或多行的音频设备。
    Port
    端口是用于输入或输出音频到音频设备的简单线。
    SourceDataLine
    源数据线是可以写入数据的数据线。
    TargetDataLine
    目标数据线是 DataLine的类型， DataLine可以读取音频数据。
  • 类摘要  
    类 描述
    AudioFileFormat
    AudioFileFormat类的一个实例描述了一个音频文件，包括文件类型，文件的长度（以字节为单位），文件中包含的音频数据的采样帧长度以及音频数据的格式。
    AudioFileFormat.Type
    Type类的一个实例代表音频文件的标准类型之一。
    AudioFormat
    AudioFormat是指定声音流中数据的特定布置的类。
    AudioFormat.Encoding
    Encoding类命名用于音频流的特定数据表示类型。
    AudioInputStream
    音频输入流是具有指定音频格式和长度的输入流。
    AudioPermission
    AudioPermission类代表对音频系统资源的访问权限。
    AudioSystem
    AudioSystem类作为采样音频系统资源的入口点。
    BooleanControl
    A BooleanControl提供了在影响线路音频的两种可能设置之间进行切换的功能。
    BooleanControl.Type
    BooleanControl.Type类的实例识别一种布尔控件。
    CompoundControl
    甲 CompoundControl ，如图形均衡器，提供了在两个或多个相关的性能，其中的每一个本身是表示作为控制 Control 。
    CompoundControl.Type
    CompoundControl.Type内部类的实例识别一种复合控件。
    Control
    Lines经常有一组控制，如增益和平移，影响通过线路的音频信号。
    Control.Type
    Type类的一个实例表示控件的类型。
    DataLine.Info
    除了从其超类继承的类信息， DataLine.Info还提供了特定于数据行的附加信息。
    EnumControl
    A EnumControl提供对一组离散可能值的控制，每个值由对象表示。
    EnumControl.Type
    EnumControl.Type内部类的一个实例标识一种枚举控件。
    FloatControl
    A FloatControl对象提供对一系列浮点值的控制。
    FloatControl.Type
    FloatControl.Type内部类的实例识别一种浮动控件。
    Line.Info
    A Line.Info对象包含有关行的信息。
    LineEvent
    LineEvent类封装了线路打开，关闭，启动或停止时线路发送监听器的信息。
    LineEvent.Type
    LineEvent.Type内部类标识一行发生的事件。
    Mixer.Info
    Mixer.Info类代表有关音频混音器的信息，包括产品的名称，版本和供应商以及文本描述。
    Port.Info
    Port.Info类扩展了 Line.Info ，具有特定于端口的附加信息，包括端口的名称以及它是混合器的源还是目标。
    ReverbType
    ReverbType类提供访问各种混响设置以应用于音频信号的方法。
  • 异常摘要  
    异常 描述
    LineUnavailableException
    A LineUnavailableException是一个例外，指示由于不可用而无法打开该行。
    UnsupportedAudioFileException
    UnsupportedAudioFileException是一个例外，指示操作失败，因为文件不包含已识别的文件类型和格式的有效数据。
