    public interface QueryExp
extends Serializable

    表示与数据库查询“where子句”类似的关系约束。 QueryExp的实例由Query类的静态方法返回。

    通过实现此接口来创建自定义查询是可能的，但不推荐使用。 在这种情况下，最好是延长QueryEval类，而不是直接实现该接口，使setMBeanServer(javax.management.MBeanServer)法正常工作。

    1.5
