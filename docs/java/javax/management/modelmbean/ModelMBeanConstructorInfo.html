    public class ModelMBeanConstructorInfo

    ModelMBeanConstructorInfo对象描述了ModelMBean的构造函数。 它是MBeanConstructorInfo的子类，添加了一个相关的描述符和DescriptorAccess接口的实现。

    描述符中的字段被定义为但不限于以下内容。 请注意，当该表中的Type为Number时，也可以使用Long的十进制表示形式的String。

    Name Type Meaning name String Constructor name. descriptorType String Must be "operation". role String Must be "constructor". displayName String Human readable name of constructor. visibility Number 1-4 where 1: always visible 4: rarely visible. presentationString String XML formatted string to describe how to present operation

    persistPolicy和currencyTimeLimit字段对构造函数是无意义的，但不被视为无效。

    默认的描述将有name ， descriptorType ， displayName和role领域。
