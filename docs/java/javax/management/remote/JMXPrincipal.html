    JMX Remote API的远程客户端的身份。

    校长如此JMXPrincipal可以与特定关联Subject来补充Subject与另外的身份。 有关如何实现这一点的更多信息，请参阅Subject课程。 授权决定可以基于与Subject相关联的Subject 。
