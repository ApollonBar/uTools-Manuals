    对缓冲通知的查询结果。 通知缓冲区中的通知具有正，单调递增的序列号。 通知查询的结果包含以下元素：

    • 最早通知的序列号仍在缓冲区中。
    • 可用于查询的下一个通知的序列号。 这将是下一个通知查询的起始序列号。
    • 对应于返回的通知和他们对应的侦听器的（Notification，listenerID）对的数组。

    nextSequenceNumber可能小于earliestSequenceNumber 。 这表示两者之间的通知可能已经丢失。
