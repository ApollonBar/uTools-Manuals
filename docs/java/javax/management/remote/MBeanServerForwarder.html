Interface MBeanServerForwarder

  • All Superinterfaces:
    MBeanServer ， MBeanServerConnection 

    public interface MBeanServerForwarder
extends MBeanServer

    此类的一个对象实现MBeanServer接口并包装另一个也实现该接口的对象。 通常，在将方法转发到包装对象之前和/或之后，此接口的实现在MBeanServer接口的某些或所有方法中执行某些操作。 示例包括安全检查和日志记录。

    从以下版本开始：
    1.5
