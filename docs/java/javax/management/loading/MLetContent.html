      • MLetContent

        public MLetContent(URL url,
                   Map<String,String> attributes,
                   List<String> types,
                   List<String> values)
        创建一个 MLet实例，使用从MLet文本文件中的 MLET标记读取的 MLET 。
        参数
        url - 包含 MLET标签的MLet文本文件的URL。
        attributes - MLET标签的MLET 。 该地图中的键是小写的属性名称，例如codebase 。 值是关联的属性值。
        types - 在嵌套的<PARAM>标签中出现的TYPE属性的列表。
        values - 在嵌套的<PARAM>标签中出现的VALUE属性的列表。
