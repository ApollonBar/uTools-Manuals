      • setMBeanServer

        public void setMBeanServer(MBeanServer s)

        设置要执行查询的MBean服务器。 该设置对执行集合的线程有效。 它被复制到创建时线程创建的任何线程。

        由于历史原因，此方法不是静态的，而是其行为不依赖于它所调用的实例。

        参数
        s - 要执行查询的MBean服务器。
        另请参见：
        getMBeanServer()

      • getMBeanServer

        public static MBeanServer getMBeanServer()

        通过此线程返回最近提供给setMBeanServer方法的MBean服务器。 如果这个线程从未调用过该方法，结果就是父线程在创建此线程时从getMBeanServer获取的值，如果没有父线程，则为null。

        结果
        MBean服务器。
