Package javax.transaction

包含ORB机械在解组期间抛出的三个异常。

See: 描述

  • 异常摘要  
    异常 描述
    InvalidTransactionException
    此异常表示请求携带无效的事务上下文。
    TransactionRequiredException
    此异常指示请求携带空事务上下文，但目标对象需要激活事务。
    TransactionRolledbackException
    此异常表示与请求的处理关联的事务已回滚，或标记为回滚。
