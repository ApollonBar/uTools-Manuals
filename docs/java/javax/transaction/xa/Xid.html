Interface Xid


  • public interface Xid
    Xid接口是X / Open事务标识符XID结构的Java映射。 此接口指定三个访问器方法来检索全局事务格式ID，全局事务ID和分支限定符。 Xid接口由事务管理器和资源管理器使用。 该界面对于应用程序是不可见的。
