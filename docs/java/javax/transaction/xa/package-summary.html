Package javax.transaction.xa Description

提供定义事务管理器和资源管理器之间的契约的API，这允许事务管理器在JTA事务中引用和删除资源对象（由资源管理器驱动程序提供）。 特定资源管理器的驱动程序供应商提供了该API的实现。

从以下版本开始：
1.4
