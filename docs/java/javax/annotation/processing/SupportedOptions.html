Annotation Type SupportedOptions


  • @Documented
 @Target(value=TYPE)

public @interface SupportedOptions
    用于指示注释处理器支持哪些选项的注释。 Processor.getSupportedOptions()方法可以从该注释的值构造其结果，如AbstractProcessor.getSupportedOptions()所做 。 只能使用strings conforming to the grammar作为值。
