      • getElementsAnnotatedWith

        Set<? extends Element> getElementsAnnotatedWith(TypeElement a)
        返回使用给定注释类型注释的元素。 注释可以直接显示或继承。 仅返回本轮注释处理中包含的包元素和类型元素，或返回在其中声明的成员，构造函数，参数或类型参数的声明。 包含的类型元素是root types和嵌套在其中的任何成员类型。 包中的元素不被认为包含在内，只因为该包的一个package-info文件被创建。
        参数
        a - 请求的注释类型
        结果
        用给定注释类型注释的元素，如果没有，则为空集
        异常
        IllegalArgumentException - 如果参数不表示注释类型

      • getElementsAnnotatedWith

        Set<? extends Element> getElementsAnnotatedWith(类<? extends Annotation> a)
        返回使用给定注释类型注释的元素。 注释可以直接显示或继承。 仅返回本轮注释处理中包含的包元素和类型元素，或返回在其中声明的成员，构造函数，参数或类型参数的声明。 包含的类型元素是root types ，嵌套在其中的任何成员类型。 包中的元素不被认为包含在内，因为该包的package-info文件已创建。
        参数
        a - 请求的注释类型
        结果
        用给定注释类型注释的元素，如果没有，则为空集
        异常
        IllegalArgumentException - 如果参数不表示注释类型
