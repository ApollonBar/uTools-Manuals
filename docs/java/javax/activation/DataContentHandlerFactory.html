javax.activation

Interface DataContentHandlerFactory


  • public interface DataContentHandlerFactory
    该界面定义了一个工厂为DataContentHandlers 。 此接口的实现应将MIME类型映射到DataContentHandler的实例。 实现此接口的类的设计模式与java.net.URL中使用的java.net.URL 。
    从以下版本开始：
    1.6
