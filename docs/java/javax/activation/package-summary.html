Package javax.activation

  • 接口摘要  
    接口 描述
    CommandObject
    Activation Framework感知的JavaBeans组件实现此界面，以找出要求执行的命令动词，并获取表示他们应该操作的数据的DataHandler。
    DataContentHandler
    DataContentHandler接口由可用于扩展DataHandler实现可传输接口的功能的对象实现。
    DataContentHandlerFactory
    该界面定义了一个工厂为 DataContentHandlers 。
    DataSource
    DataSource接口为JavaBean激活框架提供了任意数据集合的抽象。
  • 类摘要  
    类 描述
    ActivationDataFlavor
    该ActivationDataFlavor将类是一种特殊的子类 java.awt.datatransfer.DataFlavor 。
    CommandInfo
    CommandMap实现使用CommandInfo类来描述命令请求的结果。
    CommandMap
    CommandMap类为系统中可用的命令对象的注册表提供了一个接口。
    DataHandler
    DataHandler类为许多不同来源和格式的数据提供了一致的界面。
    FileDataSource
    FileDataSource类实现一个封装文件的简单DataSource对象。
    FileTypeMap
    FileTypeMap是一个抽象类，为文件提供数据输入界面。
    MailcapCommandMap
    MailcapCommandMap扩展了CommandMap抽象类。
    MimeType
    多功能Internet邮件扩展（MIME）类型，如RFC 2045和2046中所定义。
    MimeTypeParameterList
    RFC 2045和2046中定义的MimeType的参数列表。
    MimetypesFileTypeMap
    该类扩展了FileTypeMap，并通过其文件扩展名提供了文件的数据输入。
    URLDataSource
    URLDataSource类提供了一个在 URL接口中包装 URL对象的对象。
  • 异常摘要  
    异常 描述
    MimeTypeParseException
    封装MimeType解析相关异常的类。
    UnsupportedDataTypeException
    表示请求的操作不支持请求的数据类型。
