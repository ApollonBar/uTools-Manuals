    public class URLDataSource
extends Object
implements DataSource
    URLDataSource类提供了一个在URL接口中包装URL对象的对象。 URLDataSource简化了JavaBeans激活框架内由URL描述的数据的处理，因为此类可用于创建新的DataHandler。 注意：DataHandler对象使用URL构造时，内部将创建一个URLDataSource。
    从以下版本开始：
    1.6

      • getContentType

        public String getContentType()
        返回URL内容类型头字段的值。 在检索URLConnection对象后，它调用URL的URLConnection.getContentType方法。 注意：此方法尝试在URL上调用openConnection方法。 如果此方法失败，或者如果内容类型未从URLConnection返回，则getContentType返回“application / octet-stream”作为内容类型。
        Specified by:
        getContentType在界面 DataSource
        结果
        内容类型。
