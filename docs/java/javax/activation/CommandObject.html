javax.activation

Interface CommandObject


  • public interface CommandObject
    Activation Framework感知的JavaBeans组件实现此界面，以找出要求执行的命令动词，并获取表示他们应该操作的数据的DataHandler。 也可以使用不实现此接口的JavaBean。 这样的命令可以使用Externalizable接口获取数据，或使用特定于应用的方法。

    从以下版本开始：
    1.6

        void setCommandContext(String verb,
                       DataHandler dh)
                throws IOException
        使用要处理的动词来初始化命令，以及描述将要操作的数据的DataHandler。 注意：呼叫者可以通过null作为DataHandler的值。
        参数
        verb - 此对象引用的命令动词。
        dh - DataHandler。
        异常
        IOException
