    一个 SSLEngineResult枚举描述了这个 SSLEngine的当前握手状态。
    从以下版本开始：
    1.5

    • Enum Constant Summary

      Enum Constants  
      Enum Constant and Description
      FINISHED
      SSLEngine刚刚完成握手。
      NEED_TASK
      SSLEngine需要一个（或多个）委托任务的结果，握手之前可以继续。
      NEED_UNWRAP
      在 SSLEngine需要从远程端接收数据。
      NEED_WRAP
      SSLEngine必须在 SSLEngine之前将数据发送到远程端才能继续，所以应该调用 SSLEngine.wrap() 。

      SSLEngine目前不是握手。
