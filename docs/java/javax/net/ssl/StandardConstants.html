        public static final int SNI_HOST_NAME
        代表服务器名称指示（SNI）扩展名中的DNS主机名（见SNIHostName ）的“host_name”类型。

        SNI扩展是扩展SSL / TLS协议以指示客户端在握手期间尝试连接的服务器名称的功能。 见第3节，“服务器名称指示”中， TLS Extensions (RFC 6066) 。

        该常数的值为0。
