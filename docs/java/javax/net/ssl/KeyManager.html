    public interface KeyManager
    这是JSSE密钥管理器的基础接口。

    KeyManager负责管理用于向其对等体认证本地SSLSocket的密钥材料。 如果没有密钥资料可用，套接字将无法显示身份验证凭证。

    KeyManager是由使用KeyManagerFactory或通过实现一个KeyManager子类创建的。
