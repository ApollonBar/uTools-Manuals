    public class SSLSessionBindingEvent
extends EventObject
    此事件传播到SSLSessionBindingListener。 当侦听器对象绑定或绑定到SSLSession SSLSession.putValue(String, Object)或SSLSession.removeValue(String)时，实现SSLSessionBindingListener的对象将会收到此类型的事件。 事件的name字段是侦听器被绑定或解除绑定的关键。
    从以下版本开始：
    1.4
