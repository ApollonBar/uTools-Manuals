    public interface TrustManager
    这是JSSE信托管理器的基础接口。

    TrustManager负责管理在作出信任决定时使用的信任资料，并且决定是否应接受对等体呈现的凭据。

    TrustManager S被要么使用创建TrustManagerFactory ，或通过实现之一TrustManager子类。
