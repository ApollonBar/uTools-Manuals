    public interface HandshakeCompletedListener
extends EventListener
    该接口由希望接收关于在给定SSL连接上完成SSL协议握手的通知的任何类实现。

    当SSL握手完成后，新的安全参数将被建立。 这些参数始终包括用于保护消息的安全密钥。 它们还可以包括与新会话相关联的参数，例如经认证的对等体身份和新的SSL加密套件。
