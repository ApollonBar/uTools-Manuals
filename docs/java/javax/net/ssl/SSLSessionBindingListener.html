    public interface SSLSessionBindingListener
extends EventListener
    该接口由想要知道何时被绑定或从SSLSession绑定的对象来实现。 当通过SSLSession.putValue(String, Object)或SSLSession.removeValue(String)发生任一事件时，通过标识会话的SSLSessionBindingEvent来传送事件。
    从以下版本开始：
    1.4
