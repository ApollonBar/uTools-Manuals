Interface X509KeyManager

  • All Superinterfaces:
    KeyManager
    所有已知实现类：
    X509ExtendedKeyManager 

    public interface X509KeyManager
extends KeyManager
    此接口的实例管理哪些基于X509证书的密钥对用于认证安全套接字的本地端。

    在安全套接字协商期间，这个接口的方法就是调用：

    • 根据呈现的标准确定可用于谈判的一组别名，
    • 根据呈现的标准选择最佳别名 ，以及
    • 获取给定别名的相应密钥材料。

    注意：X509ExtendedKeyManager应该用于这个类。

    从以下版本开始：
    1.4
