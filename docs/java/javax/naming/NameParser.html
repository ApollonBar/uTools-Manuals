Interface NameParser


  • public interface NameParser
    此接口用于从分层命名空间中解析名称。 NameParser包含解析名称所需的句法信息（如从左到右方向，名称分隔符等）的知识。 equals（）方法当用于比较两个NameParsers时，如果且仅当它们提供相同的命名空间时才返回true。
    从以下版本开始：
    1.3
