    public class LinkException
extends NamingException
    此异常用于描述解决链接时遇到的问题。 添加信息被添加到基本的NamingException以精确定位该链接的问题。

    类似于NamingException如何捕获名称解析信息，LinkException捕获“链接”-name解析信息，以确定解决链接时遇到的问题。 所有这些字段可能为空。

    • 链接已解析名称。 已解决的链接名称部分。
    • 链接解决对象。 链接名称解析的对象。
    • 链接剩余名称。 尚未解决的链接名称部分。
    • 链接说明。 详细解释为什么链路分辨率失败。

    LinkException实例与并发多线程访问不同步。 尝试访问和修改单个LinkException实例的多个线程应锁定该对象。
