    public interface UnsolicitedNotificationListener
extends NamingListener
    此界面用于处理UnsolicitedNotificationEvent 。 “非请求通知”在RFC 2251中定义。 它允许服务器向客户端发送未经请求的通知。 A UnsolicitedNotificationListener必须：
    1. 实现这个接口及其方法
    2. 实现NamingListener.namingExceptionThrown() ，以便在尝试收集未经请求的通知事件时，会收到异常抛出的通知。
    3. 使用EventContext或EventDirContext中的addNamingListener()方法之一注册上下文。 只有这些方法的NamingListener论证是适用的; 对于UnsolicitedNotificationListener ，其余的被忽略。 （如果实现其他监听器接口，这些参数可能适用于侦听器）。
    从以下版本开始：
    1.3
