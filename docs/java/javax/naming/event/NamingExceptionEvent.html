    public class NamingExceptionEvent
extends EventObject
    此类表示烧制时用于收集信息通知的NamingEvent小号听众的程序/进程抛出NamingException的事件。 这可能发生，例如，如果监听器使用的服务器在addNamingListener()调用之后中止。
    从以下版本开始：
    1.3
