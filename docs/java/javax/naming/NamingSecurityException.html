    public abstract class NamingSecurityException
extends NamingException
    这是Context和DirContext接口中的操作抛出的与安全相关异常的超类。 故障的性质由子类名称描述。

    如果程序想要特别处理这个异常，那么在尝试捕获NamingException之前，它应该明确地捕获NamingSecurityException。 程序可能想要这样做，例如，如果它想要特别从其他类型的命名异常处理与安全相关的异常。

    适用于NamingException的同步和序列化问题直接在这里应用。
