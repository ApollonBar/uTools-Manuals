    public class StringRefAddr
extends RefAddr
    此类表示通信端点的地址的字符串形式。 它包括描述通信机制的类型和特定于该通信机制的字符串内容。 地址类型和地址内容的格式和解释基于三方的协议：使用地址的客户端，可以使用该地址达到的对象/服务器以及创建该地址的管理员或程序地址。

    字符串引用地址的示例是主机名。 字符串引用地址的另一个例子是一个URL。

    字符串引用地址是不可变的：一旦创建，它将无法更改。 对单个StringRefAddr的多线程访问不需要同步。
