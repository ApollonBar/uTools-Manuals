    public class InvalidSearchFilterException
extends NamingException
    当搜索过滤器的规范无效时抛出此异常。 过滤器的表达式可能无效，或者传递给过滤器的参数之一可能存在问题。

    适用于NamingException的同步和序列化问题直接在这里应用。
