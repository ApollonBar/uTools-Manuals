      • getRunTimeCodeBase

        RunTime getRunTimeCodeBase()
        返回此ValueHandler的CodeBase。 这被ORB运行时使用。 服务器在第一个GIOP回复中发送包含该CodeBase的IOR的服务上下文。 客户端在第一个GIOP请求中执行相同操作。
        结果
        该ValueHandler的SendingContext.CodeBase。
