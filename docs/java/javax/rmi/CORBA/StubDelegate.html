  • public interface StubDelegate
    支持Stub中方法实现的委派 。 委托是实现此接口的类的实例，并为javax.rmi.CORBA.Stub的所有方法提供了替换javax.rmi.CORBA.Stub 。 如果启用委派，则每个存根具有关联的委托。 通过提供委托人的类名作为javax.rmi.CORBA.StubClass系统属性的值来启用代表。
