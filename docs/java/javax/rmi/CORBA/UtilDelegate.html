javax.rmi.CORBA

Interface UtilDelegate


  • public interface UtilDelegate
    支持Util中的方法实现委派 。 委托是实现此接口的类的单例实例，并为javax.rmi.CORBA.Util的所有方法提供javax.rmi.CORBA.Util 。 通过提供委托人的类名作为javax.rmi.CORBA.UtilClass系统属性的值来启用委派。
