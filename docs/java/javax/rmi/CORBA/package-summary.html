Package javax.rmi.CORBA Description

包含RMI-IIOP的可移植API。 这些API在生成的存根和关系和RMI-IIOP运行时间之间提供了标准接口。 他们还允许第三方ORB用作ROP超过IIOP，作为Sun提供的ORB的替代品。 它们不是直接从RMI-IIOP应用中调用的。 另请参见javax.rmi包。
