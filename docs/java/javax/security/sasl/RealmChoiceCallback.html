        public RealmChoiceCallback(String prompt,
                           String[] choices,
                           int defaultChoice,
                           boolean multiple)
        构造一个 RealmChoiceCallback与提示，选择列表和默认选择。
        参数
        prompt - 用于请求领域的非空提示。
        choices - 可以选择的领域的非空列表。
        defaultChoice - 显示选项列表时，选择作为默认选择。 它是choices阵列的索引。
        multiple - 如果允许多个选项，则为true; 否则为假
        异常
        IllegalArgumentException -如果 prompt为空或空字符串，如果 choices具有为0的长度，如果来自任何元件 choices是空值或空白，或者如果 defaultChoice不落在阵列边界内 choices
