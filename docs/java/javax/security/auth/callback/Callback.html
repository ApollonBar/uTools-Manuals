    public interface Callback

    该接口的实现被传递到CallbackHandler ，允许底层安全服务与呼叫应用程序交互以检索特定认证数据（例如用户名和密码）或显示某些信息（例如错误和警告消息）的能力。

    Callback实现不检索或显示底层安全服务请求的信息。 Callback实现简单地提供了将这些请求传递给应用程序的方法，如果适用，应用程序将请求的信息返回到底层安全服务。

    另请参见：
    CallbackHandler ， ChoiceCallback ， ConfirmationCallback ， LanguageCallback ， NameCallback ， PasswordCallback ， TextInputCallback ， TextOutputCallback
