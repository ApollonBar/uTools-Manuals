      • PasswordCallback

        public PasswordCallback(String prompt,
                        boolean echoOn)
        构造一个 PasswordCallback ，带有一个提示符和一个布尔值，指定在输入密码时是否显示该密码。

        参数
        prompt - 用于请求密码的提示。

        echoOn - 如果密码在输入时 echoOn ，则为true。
        异常
        IllegalArgumentException - 如果 prompt为空，或者如果 prompt的长度为0。
