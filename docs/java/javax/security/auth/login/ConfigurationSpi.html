    这个类定义的 （SPI） Configuration类。 此类中的所有抽象方法必须由希望提供配置实现的每个服务提供商实现。

    此抽象类的子类实现必须提供一个公共构造函数，它将一个Configuration.Parameters对象作为输入参数。 如果不了解Configuration.Parameters输入，此构造函数也必须引发Configuration.Parameters 。

      • engineGetAppConfigurationEntry

        protected abstract AppConfigurationEntry[] engineGetAppConfigurationEntry(String name)
        检索指定 名称的AppConfigurationEntries。

        参数
        name - 用于索引配置的名称。
        结果
        指定 名称的AppConfigurationEntries数组，如果没有条目，则为null。

      • engineRefresh

        protected void engineRefresh()
        刷新并重新加载配置。

        此方法使此Configuration对象以实现相关的方式刷新/重新加载其内容。 例如，如果此配置对象将其条目存储在文件中，则调用refresh可能会导致重新读取该文件。

        这种方法的默认实现什么都不做。 如果实现支持刷新操作，则该方法应该被覆盖。

        异常
        SecurityException - 如果主叫方没有刷新其配置的权限。
