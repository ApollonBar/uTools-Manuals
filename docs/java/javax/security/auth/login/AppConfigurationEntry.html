  • public class AppConfigurationEntry
extends Object
    此类表示为Configuration类中的getAppConfigurationEntry(String appName)方法中指定的应用Configuration配置的单个LoginModule条目。 每个各自的AppConfigurationEntry包含一个LoginModule名称，一个控制标志（指定此LoginModule是否需要，REQUISITE，SUFFICIENT或可选）和LoginModule特定的选项。 有关不同控制标志及其语义的更多信息，请参考Configuration类。
