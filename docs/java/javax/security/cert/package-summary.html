Package javax.security.cert Description

提供公钥证书类。 这些类包括java.security.cert包的简化版本。 这些类是作为Java安全套接字扩展（JSSE）的一部分开发的。 当JSSE被添加到J2SE版本1.4中时，仅为了向后兼容的原因添加了该包。 新的应用程序不应该使用这个包，而是使用java.security.cert。
从以下版本开始：
1.4
